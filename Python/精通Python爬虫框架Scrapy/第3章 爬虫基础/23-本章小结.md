### 3.5　本章小结

本章可能是大家开始学习Scrapy时最重要的一章。你刚刚学习了开发爬虫最基本的方法：UR<sup class="my_markdown">2</sup>IM。你学会了如何自定义适合需求的 `Item` ，使用 `ItemLoader` 、XPath表达式和处理器加载 `Item` ，以及如何对 `Request` 使用 `yield` 操作。我们使用 `Request` 横向到达不同的索引页，纵向到达房源页并抽取 `Item` 。最后，我们看到了如何使用 `CrawlSpider` 和 `Rule` ，以很少的代码行创建非常强大的爬虫。如果你想要更深入地理解这些概念，请尽可能多地阅读本章，当然，也可以在你开发自己的爬虫时使用本章作为参考。
