### 1. while

代码：

```python
name_list = ['Tom', 'Lily', 'Rose']

i = 0
while i < len(name_list):
    print(name_list[i])
    i += 1
```

执行结果：

```console
Tom
Lily
Rose
```

### 2. for

代码：

```python
name_list = ['Tom', 'Lily', 'Rose']

for i in name_list:
    print(i)
```

执行结果：

```console
Tom
Lily
Rose
```
