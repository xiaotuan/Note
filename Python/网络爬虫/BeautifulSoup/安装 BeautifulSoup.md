BeautifulSoup 4 的所有安装方法都在 <http://www.crummy.com/software/BeautifulSoup/bs4/doc/> 里面。

+ Linux 系统安装方法

  ```shell
  $ sudo apt-get install python-bs4
  ```

+ Mac 系统安装方法

  1. 先安装 Python 的包管理器 pip

     ```shell
     $ sudo easy_install pip
     ```

  2. 在使用 pip 安装 BeautifulSoup 4

     ```shell
     $ pip install beautifulsoup4
     ```

+ Windows 系统安装方法

  1. 先安装 Python 的包管理费 pip

     从 <https://bootstrap.pypa.io/get-pip.py> 下载 get-pip.py 文件，再执行下面命令：

     ```shell
     > python get-pip.py
     ```

  2. 再使用 pip 安装 BeautifulSoup 4

     ```shell
     > python -m pip install beautifulsoup4
     ```
