为了找到网站的所有者，我们可以使用 WHOIS 协议查询域名的注册者是谁。Python 中有一个针对该协议的封装库，其文档地址为 <https://pypi.python.org/pypi/python-whois>，我们可以通过 pip 进行安装：

```console
$ pip install python-whois
```

下面是使用该模块对 appspot.com 这个域名进行 WHOIS 查询是的返回结果。

