### 18.3.4　登录到SMTP服务器

到SMTP服务器的加密连接建立后，可以调用 `login()` 方法，用你的用户名（通常是你的电子邮件地址）和电子邮件口令登录。

```javascript
>>> smtpObj.login('my_email_address@example.com', 'MY_SECRET_PASSWORD')
(235, b'2.7.0 Accepted')
```

传入电子邮件地址字符串作为第一个参数，传入口令字符串作为第二个参数。返回值 `235` 表示认证成功。如果口令不正确，Python会抛出 `smtplib.`  `SMTPAuthenticationError` 异常。



**警告：**
将口令放在源代码中要当心。如果有人复制了你的程序，他们就能访问你的电子邮件账户！调用 `input()` ，让用户输入口令是一个好主意。每次运行程序时输入口令可能不方便，但这种方法不会在未加密的文件中留下你的口令，其他人不会轻易地得到它。
