点号或者句点（`.`）符号匹配除了换行符 `\n` 以外的任何字符（`Python` 正则表达式有一个编译标记 `[S或者DOTALL]` ，该标记能够推翻这个限制，使点号能够匹配换行符）。

| 正则表达式模式 | 匹配的字符串                                                 |
| -------------- | ------------------------------------------------------------ |
| `f.o`          | 匹配在字母 `f` 和 `o` 之间的任意一个字符；例如 `fao`、`f9o`、`f#o` 等 |
| `..`           | 任意两个字符                                                 |
| `.end`         | 匹配在字符串 `end` 之前的任意一个字符                        |

> 提示：要显示匹配一个句点符号本身，可以使用反斜杠转义句点符号 `\.`。
