方括号中两个符号中间用连字符（`-`）连接，用于指定一个字符的范围；例如，`A-Z`、`a-z` 或者 `0-9` 分别用于表示大写字母、小写字母和数值数字。

| 正则表达式模式     | 匹配的字符串                                                 |
| ------------------ | ------------------------------------------------------------ |
| `z.[0-9]`          | 字母 `z` 后面跟着任何一个字符，然后跟着一个数字              |
| `[r-u][env-y][us]` | 字母 `r`、`s`、`t` 或者 `u` 后面跟着 `e`、`n`、`v`、`w`、`x` 或者 `y`，然后跟着 `u` 或者 `s` |
| `[^aeiou]`         | 一个非元音字符                                               |
| `[^\t\n]`          | 不匹配制表符或者换行符                                       |
| `["-a]`            | 在一个 `ASCII` 系统中，所有字符都位于 `"` 和 `a` 之间，即 34 ~ 97 之间 |
