方括号正则表达式能够匹配一对方括号中包含的任何字符。

| 正则表达式模式     | 匹配的字符串                                                 |
| ------------------ | ------------------------------------------------------------ |
| `\b[aeiu]t`        | bat、bet、bit、but                                           |
| `[cr][23][dp][o2]` | 一个包含四个字符的字符串，第一个字符是 `c` 或 `r`，然后是 `2` 或 `3`，后面是 `d` 或 `p`，最后要么是 `o` 要么是 `2`。例如 c2do、r3p2、r2d2、c3po 等 |
