### 1. 数值

#### 1.1 int（整型）

```python
# int -- 整型
num1 = 1
print(type(num1))
```

#### 1.2 float（浮点型）

```python
# float -- 浮点数，就是小数
num2 = 1.1
print(type(num2))
```

### 2. 布尔型

#### 2.1 True（真）

```python
# bool -- 布尔型，通常判断使用，布尔型有两个取值 True 和 False
b = True
print(type(b))
```

#### 2.2 False（假）

```python
# bool -- 布尔型，通常判断使用，布尔型有两个取值 True 和 False
b = False
print(type(b))
```

### 3. 字符串

```python
# str -- 字符串型
a = "Hello"
print(type(a))
```

### 4. 列表

```python
# list -- 列表
c = [10, 20, 30]
print(type(c))
```

### 5. 元组

```python
# tuple -- 元组
d = (10, 20, 30)
print(type(d))
```

### 6. 集合

```python
# set -- 集合
e = {10, 20, 30}
print(type(e))
```

### 7. 字典

```python
# dict -- 字典
f = {'name': 'TOM', 'age': 18}
print(type(f))
```
