在 Python 中，变量不需要声明变量的数据类型，变量可以存储任何数据。例如：

```console
xiaotuan@xiaotuan:~$ python
Python 3.5.2 (default, Oct  7 2020, 17:19:02) 
[GCC 5.4.0 20160609] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>> message = "Hello Python world!"
>>> print(message)
Hello Python world!
>>> message = 345
>>> print(message)
345
>>> 
```

上面的例子定义了一个 message 变量并将一个字符串赋给它，然后又将一个整数赋给 message，程序可以正常运行。
