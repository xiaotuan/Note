### 1. and 操作符

`and` 操作符总是接收两个布尔值（或表达式），如果两个布尔值都为 `True`，表达式的值为 `True`，否则为 `False`。例如：

```python
>>> True and True
True
>>> True and False
False
```

### 2. or 操作符

`or` 操作符总是接收两个布尔值（或表达式），只要其中一个布尔值为 `True`，则 `or` 表示式的值就为 `True`，否则为 `False`。例如：

```python
>>> False or True
True
>>> False or False
False
```

### 3. not 操作符

`not` 操作符只作用于一个布尔值（或表达式），`not` 操作符求值为相反的布尔值：

```python
>>> not True
False
>>> not False
True
```
