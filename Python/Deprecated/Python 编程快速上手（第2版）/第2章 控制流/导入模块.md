### 1. import 语句

可以使用 `import` 语句导入模块，格式如下：

```python
import 模块名
```

例如：

```python
import random

for i in range(5):
    print(random.randint(1, 10))
```

也可以使用 `import` 语句同时导入多个模块，格式如下：

```python
import 模块名1, 模块名2, 模块名3
```

例如：

```python
import random, sys, os, math
```
