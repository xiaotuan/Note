### 1. 开发工具的安装

开发工具可以直接到微软的 Windows 开发者网站进行下载（<http://dev.windows.com>）。

Windows 10 Developer Tools 包含的工具集合详细信息如下：

+ Visual Studio

  Visual Studio 是 Windows 10 的继承开发环境，其中包括了 C# 和 XAML 代码编辑功能、简单界面的布局与设计功能、编译程序、连接模拟器、部署程序，以及调试程序等功能。

+ Emulator

  Emulator 是 Windows 10 的模拟器，开发者可以把应用程序部署到各种分辨率的模拟器上来检查程序运行的效果。

+ Blend for Visual Studio

  Blend for Visual Studio 是强大的 XAML UI 设计工具，使用 Expression Blend 可以补全 Visual Studio 缺乏的更加强大的 UI 设计功能。
