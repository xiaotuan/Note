<center> Access 各个字段类型的使用情形及其所占用的存储空间</center>

| 字段类型  | 使用情形                                                     | 占用的存储空间                                               |
| --------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 文本      | 数据包含的是文本、文本与数字的组合或是不用于算式中的数字。这样的例子有名字、地址、部门代号和电话号码 | 根据字段中所存储的内容不同，它可以是 0 ~ 255 个字节          |
| 备注      | 较长的文本和数字字符串。这样的例子有注释和说明               | 可以是 0 ~ 65536 个字节                                      |
| 数字      | 这样的数据要包含到算式当中（金钱数除外）。这样的例子有年龄、代号（如员工 ID）和付款方式 | 根据所选择字段的大小不同，它可以是 1，2，4 或 8 个字节（对复制 ID 而言，可以是 16 个字节） |
| 日期/时间 | 日期和时间。这样的例子有订货日期和出生日期                   | 8 个字节                                                     |
| 货币      | 货币值。这样的例子有应付款数和价格                           | 8 个字节                                                     |
| 自动编号  | 单一的顺序号或随机号。这样的例子有发票号和工程号             | 4 个字节（对复制 ID 而言，可以是 16 个字节）                 |
| 是/否     | 这样的字段包含两个值（是/否，对/错）。这样的例子有账单支付和占有情况 | 1 个字节                                                     |
| OLE 对象  | 这是指像 Word 文档和 Excel 电子表格这样的对象                | 根据字段内容的不同，它可以是 0 ~ 1G 个字节。这样的例子有雇员概览和预算 |
| 附件      | 图像、电子表格文件、文档、图表及其他被支持的文件类型         | 因字段中存储内容的不同而不同                                 |
| 超链接    | 这是文本或者文本与数字的组合，它存储为文本，但用作超链接地址或 UNC 路径。这样的例子有 Web 页和网络文件 | 组成地址的 3 个部分可以是 0 ~ 2048 个字节（加起来可达 64000 个字符） |
| 查阅向导  | 用于创建字段。允许用户通过向导定义的组合框从另一张表或一些值中选择这些字段 | 常规需要 4 个字节。它需要的空间与查阅字段主题所占的存储空间一样大 |
