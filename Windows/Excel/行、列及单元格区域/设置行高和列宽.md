### 1. 精确设置行高和列宽

设置行高前，先选定目标行（单行或者多行）整行或者行中单元格，然后在 Excel 功能区上依次单击 【开始】->【格式】->【行高】，在弹出的【行高】对话框中输入所需设定行高的具体数值，最后单击【确定】按钮完成操作。设置列宽的方法与此类似。

另一种方法是在选定行或者列后单击鼠标右键，在弹出的快捷菜单中选择【行高】（或者【列宽】）命令，然后进行相应的操作。

### 2. 直接改变行高和列宽

在工作表中选中单列或者多列，当鼠标指针放置在选中列与相邻列标签之间时，列标签之间中线上的鼠标指针显示为一个黑色双箭头。按住鼠标左键不放，向左或者向右拖动鼠标，此时在列标签上方会出现一个提示框，显示当前的列宽。调整到所需的列宽时，松开鼠标左键，即可完成列宽的设置。设置行高的方法与此操作类似。

> **行高和列宽数值的单位**
>
> 行高的单位是磅（Point）。这里的磅并非英制重量单位的磅（Pound），而是一种印刷业描述印刷字体大小的专用尺度，英文 Point 的音译，所以磅数制又被称为点制、点数制。1 磅近似等于 1/72 英寸（inch），1 英寸等于 25.4mm，所以 1 磅近似等于 0.352 78mm。行高的最大限制为 409 磅，即 144.286mm。
>
> 列宽的单位是字符。列宽的数值是指适用于单元格的 "标准字体" 的数字 0 ~ 9 的平均值。所谓的 "标准字体"，是指在【Excel 选项】对话框【常规】选项卡中【新建工作簿时】区域的【使用此字体作为默认字体】处的设置，包括了选用字体及字号。如果不考虑不同字符之间的宽度差异，可以用更通俗易懂的话来描述列宽：列宽的值表示这一列所能容纳的数字字符个数。列宽设置的数字为 0 ~ 255 之间，即列宽的最大限制为 255 个字符，最小列宽为 0，当列宽设置为 0 时，即隐藏该列。
>
> 有时，用户可能需要将行高和列宽建立一定的关系，例如需要设置出一个正方形的单元格。此时行高和列宽的不可比性形成了障碍，需要借助另一个隐形的行高列宽单位——像素（pixel）来解决。
>
> 虽然无法再菜单中以像素作为行高列宽的单位设置精确数值，但是在直接拖动鼠标设置行高列宽的过程中，像素这个隐形的宽度单位就会被显示出来。

### 3. 设置最合适的行高和列宽

选中需要调整列宽的多列，在 Excel 功能区上依次单击【开始】->【格式】->【自动调整列宽】，将选定列的列宽调整到 "最合适" 的宽度，使得列中的每一行字符都可以恰好完全地显示。

类似地，使用菜单中的【自动调整行高】命令，则可以设置最合适的行高，以适应行中字符的高度。

同时选中需要调整列宽的多列，将鼠标放置在列标签之间的中线上，此时，鼠标箭头显示为一个黑色双向箭头。双击鼠标左键，即可完成设置 "自动调整列宽" 的操作。"自动调整行高" 的方法与此类似。

### 4. 标准列宽

在 【格式】下拉菜单中，还有一条【默认列宽】命令（无 "默认行高" 命令）。使用 【默认列宽】命令，可以一次性修改当前工作表的所有列宽。但是，该命令对已经设置过列宽的列无效，也不会影响其他工作表以及新建工作表或工作簿。如果要为所有的新工作簿和工作表定义默认列宽，需要使用模板功能。
