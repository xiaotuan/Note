### 4.1.2　变量的声明和赋值

变量的声明语法如下。

```c
[访问修饰符]  [变量修饰符]  变量的数据类型  变量名
```

其中访问修饰符和变量修饰符可以省略，这将在9.2节中介绍。变量声明和赋值如下所示。

```c
01  private  int  x;       //声明x为整型变量，private可以省略
02  x=10;                  //变量的赋值，让变量x的值为10
03  string  name="John";   //声明一个字符串变量name，值为John
```

也可以在声明变量的同时赋值，如：

```c
int  x=10;
```

也可以在一行声明多个变量，如：

```c
int x=3, y=8;   //不同变量之间用逗号隔开
```
