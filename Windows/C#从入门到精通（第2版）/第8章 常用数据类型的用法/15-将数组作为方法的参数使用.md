### 8.4.3　将数组作为方法的参数使用

数组作为C#中一种常用的数据类型，可以作为参数传递给某个方法。

例如，定义方法print为：

```c
void print（int array[]）
{
…
}
```

下面的语句将数组a作为print方法的参数。

```c
int[,] a = { 1, 2, 3, 4, 5, 6 }; 
print(a);
```

类似的，如果print方法的参数是二维数组，如void print（int array[，]），则可以将一个二维数组作为参数调用print方法。当然，如果定义print方法时为数组类型参数加上ref或out修饰符，则print方法中对数组类型参数中元素的修改将会被传递回数组a的对应元素。
