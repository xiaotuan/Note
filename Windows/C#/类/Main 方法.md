`Main()` 方法是程序的入口点，程序将在此处创建对象和调用其他方法，一个 C# 程序中只能有一个入口点，每新建一个项目，程序都会自动生成一个 `Main()` 方法，默认的 `Main()` 方法代码如下所示：

```c#
static void Main(string[] args)
{
    
}
```

> 说明：`Main()` 方法默认访问级别为 `private`。

`Main()` 方法是一个特别重要的方法，使用时需要注意以下几点：

+ `Main()` 方法是程序的入口点，程序控制在该方法中开始和结束。
+ `Main()` 方法在类或结构的内部声明，它必须为静态方法，而且不应该为公共方法。
+ `Main()` 方法可以具有 `void` 或 `int` 返回类型。
+ 声明 `Main()` 方法时既可以使用参数，也可以不使用参数。
+ 参数可以作为从零开始索引的命令行参数来读取。
