在 C# 中，使用赋值运算符 "="（等号）来给变量赋值，将等号右边的值赋给左边的变量。例如：

```c#
int sum;	// 声明一个变量
sum = 2019;	// 使用赋值运算符 "=" 给变量赋值
```

在给变量赋值时，等号右边也可以是一个已经被赋值的变量。例如：

```c#
int sum, num;	// 声明两个变量
sum = 927;	// 给变量 sum 赋值为 927
num = sum;
```

> 注意：在对多个同类型的变量赋同一个值时，可以同时对多个变量进行初始化：
>
> ```c#
> int a, b, c, d, e;
> a = b = c = d = e = 0;
> ```
