`if else` 语句的语法格式如下所示：

```C#
if (布尔运算符)
{
    语句序列
} else {
    语句序列
}
```

例如：

```c#
class Program
{
    static void Main(string[] args)
    {
        int i = 927;
        if (i > 927)
        {
            Console.WriteLine("i 大于 927");
        } 
        else
        {
            Console.WriteLine("i 不大于 927");
        }
    }
}
```
