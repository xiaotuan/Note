结构是一种值类型，通常用来封装一组相关的变量，结构中可以包括构造函数、常量、字段、方法、属性、运算符、事件和嵌套类型等。但如果要同时包括上述几种成员，则应该考虑使用类。

结构具有以下特点：

+ 结构是值的类型。
+ 向方法传递结构时，结构是通过传值方式传递的，而不是作为引用传递的。
+ 结构的实例化可以不使用 `new` 运算符。
+ 结构可以声明构造函数，但它们必须带参数。
+ 一个结构不能从另一个结构或类继承。所有结构都直接继承自 `System.ValueType`，后者继承自 `System.Object`。
+ 结构可以实现接口。
+ 在结构中初始化实例字段是错误的。

> 注意：在结构声明中，除非字段被声明为 `const` 或 `static`，否则无法初始化。
