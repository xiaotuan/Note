### 1. 激活步骤

#### 1.1 查看激活状态

```bat
slmgr.vbs -xpr
```

#### 1.2 卸载秘钥

```bat
slmgr.vbs /upk
```

#### 1.3 安装秘钥

```bat
slmgr /ipk M7XTQ-FN8P6-TTKYV-9D4CC-J462D
```

#### 1.4 执行秘钥

```bat
slmgr /skms kms.xspace.in
```

或

```bat
Slmgr /skms kms.03k.org
```

#### 1.5 执行激活

```bat
slmgr /ato
```

#### 1.6 激活状态

```bat
slmgr -dlv
```
