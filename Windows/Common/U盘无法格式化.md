<center><font size="5"><b>U盘被分区出EFI，格式化失败，win无法格式化</b></font></center>

**存在问题：**
格式化不了，被莫名分区。

**报错：**
win无法格式化
u盘无法确定此卷上的扇区数量. 格式化失败.

**尝试了失败的方法：**

1. 直接右键格式化
2. 下了各种奇怪的U盘修复工具，没用。所以不要下其他的~！
3. 在分区助手里看，找不到
4. 在win10自带磁盘管理中，找不到
5. cmd中输入
   format X:/fs:fat32
   其中，X是插入U盘后，显示的所在盘、千万别打错。
   enter还是失败。
6. 换usb接口，同样失败。

**成功解决方法：**
使用diskpart.exe

```
diskpart.exe
list disk
select disk x（看你的磁盘数字）
clean
```
之后打开磁盘管理，刷新就找到了未分配的优盘
新建卷就可以了。
