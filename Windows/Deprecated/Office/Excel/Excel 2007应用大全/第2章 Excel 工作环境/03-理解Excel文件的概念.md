**表 2-1**                       							**Excel 文件格式简要说明**

| 格式                          | 扩展名 | 存储机制和限制说明                                           |
| ----------------------------- | ------ | ------------------------------------------------------------ |
| Excel 工作簿                  | .xlsx  | Excel 2007 默认的基于 XML 的文件格式。不能存储 <br/>Microsoft Visual Basic for Applications（VBA）宏代码或<br/>Microsoft Office Excel 4.0 宏工作表（.xlm） |
| Excel 启动宏的工作簿          | .xlsm  | Excel 2007 基于 XML 和启用宏的文件格式。存储 VBA 宏代码或<br/>Excel 4.0 宏工作表（.xlm） |
| Excel 二进制工作簿            | .xlsb  | Excel 2007 的二进制文件格式（BIFF12）                        |
| Excel 97 - Excel 2003 工作簿  | .xls   | Excel 97 - Excel 2003 二进制文件格式（BIFF8）                |
| XML 数据                      | .xml   | XML 数据格式                                                 |
| 单个文件网页                  | 无     | MHTML Document 文件格式                                      |
| 模板                          | .xltx  | Excel 2007 的 Excel 模板默认的文件格式。不能存储 VBA<br/>宏代码或 Excel 4.0 宏工作表（.xlm） |
| Excel 启用宏的模板            | .xltm  | Excel 模板 Excel 2007 启用宏的文件格式。存储 VBA <br/>宏代码或 Excel 4.0 宏工作表（.xlm） |
| Excel 97 - Excel 2003 模板    | .xlt   | Excel 模板的 Excel 97 - Excel 2003 二进制文件格式（BIFF8）   |
| 文本（以制表符分隔）          | .txt   | 将工作簿另存为以制表符分隔的文本文件，以便在其他<br/>Microsoft Windows 操作系统上使用，并确保正确解释制表符、<br/>换行符和其他字符。仅保存活动工作表。 |
| Unicode 文本                  | .txt   | 将工作簿另存为 Unicode 文本，这是一种由 Unicode 协会开发<br/>的字符编码标准 |
| XML 电子表格 2003             | .xml   | XML 电子表格 2003 文件格式（XMLSS）                          |
| Microsoft Excel 5.0/95 工作簿 | .xls   | Excel 5.0/95 二进制文件格式（BIFF5）                         |
| CSV （以逗号分隔）            | .csv   | 将工作簿另存为以逗号分隔的文本文件，以便在其他 Windows <br/>操作系统上使用，并确保正确解释制表符、换行符和其他字符。<br/>仅保存活动工作表。 |
| 带格式文本（以空格分隔）      | .prn   | Lotus 以空格分隔的格式。仅保存活动工作表。                   |
| DIF                           | .dif   | 数据交换格式。仅保存活动工作表。                             |
| Excel 加载项                  | .xlam  | Excel 2007 基于 XML 和启用宏的加载项格式。加载项是用于<br/>运行其他代码的补充程序。支持 VBA 项目和 Excel 4.0 宏工作表<br/>（.xlm）的使用。 |
| Excel 97 - Excel 2003 加载项  | .xla   | Excel 97 - Excel 2003 加载项，即设计用于运行其他代码<br/>的补充程序。支持 VBA 项目的使用。 |
| SYLK                          | .slk   | 符号链接格式。仅保存活动工作表                               |
