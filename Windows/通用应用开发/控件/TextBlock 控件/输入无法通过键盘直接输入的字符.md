如果需要显示无法通过键盘直接输入的字符，可以使用标准的 `XML` 转义字符来指定 `Unicode` 值。例如，如果只有一个美式键盘，但要显示 "This costs €55"，可以这样以 `Unicode` 形式指定其中的欧元符号：

```xml
<TextBlock Text="This costs &#8364;55" />
```

或者使用十六进制：

```xml
<TextBlock Text="This costs &#x20AC;55" />
```

与标准 `XML` 相同，字符串中可以包含以 `&` 符号为前缀的转义字符来表示特殊字符：

+ `&amp;` 表示 `&`
+ `&apos;` 表示单引号
+ `&quot;` 表示双引号
+ `&lt;` 表示左尖括号（小于号）
+ `&gt;` 表示右尖括号（大于号）
