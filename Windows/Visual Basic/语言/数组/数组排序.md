### 1. 选择排序

选择排序法是指每次选择所要排序的数组中值最大（由小到大排序则选择最小值）的数组元素，将这数组元素的值与前面的数组元素的值互换。

<center><b>使用选择排序法为数组 A 排序</b></center>

| <sub>排序过程</sub>    <sup>数组元素</sup> | A(1) | A(2) | A(3) | A(4) | A(5) |
| ------------------------------------------ | ---- | ---- | ---- | ---- | ---- |
| 起始值                                     | 3    | 2    | 7    | 9    | 5    |
| 第 1 次                                    | 9    | 2    | 3    | 7    | 5    |
| 第 2 次                                    | 9    | 7    | 2    | 3    | 5    |
| 第 3 次                                    | 9    | 7    | 5    | 2    | 3    |
| 第 4 次                                    | 9    | 7    | 5    | 3    | 2    |
| 排序结果                                   | 9    | 7    | 5    | 3    | 2    |

程序代码如下所示：

```vb
Private Sub CmdOk_Click(sender As Object, e As EventArgs) Handles CmdOk.Click
    Dim A(4) As Integer
    A(0) = 3 : A(1) = 2 : A(2) = 7 : A(3) = 9 : A(4) = 5
    Dim i As Long, l As Long, n As Long
    For i = 0 To 4
        For l = i + 1 To 4
            If A(i) < A(l) Then
                n = A(i)
                A(i) = A(l)
                A(l) = n
            End If
        Next
    Next
End Sub
```

### 2. 冒泡排序法

冒泡排序法是指在排序时，每次比较数组中相邻的两个数组元素的值，将较大的值排在较小的值面前。

<center><b>使用冒泡排序法为数组 A 排序</b></center>

| <sub>排序过程</sub>    <sup>数组元素</sup> | A(1) | A(2) | A(3) | A(4) | A(5) |
| ------------------------------------------ | ---- | ---- | ---- | ---- | ---- |
| 起始值                                     | 3    | 2    | 7    | 9    | 5    |
| 第 1 次                                    | 3    | 7    | 9    | 5    | 2    |
| 第 2 次                                    | 7    | 9    | 5    | 3    | 2    |
| 第 3 次                                    | 9    | 7    | 5    | 3    | 2    |
| 第 4 次                                    | 9    | 7    | 5    | 3    | 2    |
| 排序结果                                   | 9    | 7    | 5    | 3    | 2    |

程序代码如下所示：

```vb
Private Sub CmdOk_Click(sender As Object, e As EventArgs) Handles CmdOk.Click
    Dim A(4) As Integer
    A(0) = 3 : A(1) = 2 : A(2) = 7 : A(3) = 9 : A(4) = 5
    Dim i As Long, l As Long, n As Long
    Dim j As Long
    For i = 0 To 4
        For l = 0 To 3
            If A(l) < A(l + 1) Then
                n = A(l)
                A(l) = A(l + 1)
                A(l + 1) = n
            End If
        Next
        Debug.Print("第 " & (i + 1) & " 遍结果：")
        For j = 0 To 4
            Debug.Print("A(" & j & ") = " & A(j))
        Next
    Next
End Sub
```

### 3. 使用 Array.Sort() 方法排序

代码如下所示：

```vb
Private Sub SortAnimals()
    Dim zooAnimals(2) As String
    zooAnimals(0) = "lion"
    zooAnimals(1) = "turtle"
    zooAnimals(2) = "ostrich"
    Array.Sort(zooAnimals)
End Sub
```
