`Function` 过程又称函数过程。与子过程不同的是，函数过程可以返回一个值给程序调用。

### 1. 建立函数过程

使用 `Function` 语句建立函数过程的语法格式如下所示：

```vb
[Private | Public][Static] Function 函数名[(参数列表)][As 类型]
    <语句>
    [Exit Function]
    <语句>
End Function
```

`Function` 是函数过程的开始标记；`End Function` 是函数过程的结束标记；`<语句>` 是具有特定功能的程序段；`Exit Function` 语句表示退出函数过程；`As` 子句决定函数过程返回值的数据类型，如果忽略 `As` 子句，函数过程返回值的数据类型为变体型。

### 2. 调用函数过程

调用函数过程的语法格式如下所示：

```vb
变量名 = 函数名(参数列表)
```

如果没有函数名，则函数过程将返回一个默认值——数值函数返回 0；字符串函数返回一个零长度字符串，也就是空字符串；变体函数则返回 Empty。
