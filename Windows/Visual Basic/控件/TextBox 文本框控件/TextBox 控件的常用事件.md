<center><b>文本框控件的常用事件</b></center>

| 事件        | 说明                                                         |
| ----------- | ------------------------------------------------------------ |
| TextChanged | 当用户按键或将文本粘贴到文本框中时发生。使用该事件处理特定的按键（如捕捉特定的键），或者需要在内容改变时执行特定操作时使用它 |
| Click       | 当用户单击文本框时发生，当不关心鼠标指针的坐标时使用该事件   |
| MouseDown   | 当用户在文本框上按下鼠标按钮时发生，该事件通常与 MouseUp 一起使用 |
| MouseUp     | 当用户在文本框上松开鼠标按钮时发生。当 Click 事件提供的功能不能满足要求时，使用 MouseDown 和 MouseUp 事件 |
| MouseMove   | 当用户在文本框上移动鼠标时发生。使用该事件根据鼠标的移动来执行操作 |
