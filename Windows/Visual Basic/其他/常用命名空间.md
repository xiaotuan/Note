<center><b>常用命名空间</b></center>

| 命名空间              | 说明                                                         |
| --------------------- | ------------------------------------------------------------ |
| Microsoft.VisualBasic | 包含支持使用 `Visual Basic` 语言进行编译和代码生成的类       |
| System                | 包含基础类，这些类用于定义常用的值及引用数据类型、事件处理程序、接口、属性和异常。这是 `.NET` 的基础命名空间 |
| System.Data           | 包含组成 `ADO.NET` 体系结构的类                              |
| System.Diagnostics    | 包含可用于调试应用程序和跟踪代码运行的类                     |
| System.Drawing        | 包含用于访问图形设备接口（Graphical Device Interface, GDI+）基础图形功能的类 |
| System.IO             | 包含允许对数据流与文件进行读写的类                           |
| System.Net            | 包含为多种网络协议提供编程接口的类                           |
| System.Security       | 包含为公共语言运行时安全系统提供底层结构的类                 |
| System.Web            | 包含为浏览器/服务提供通信接口的类                            |
| System.Windows.Forms  | 包含用于创建基于 `Windows` 的应用程序的类，这些应用程序可以充分利用微软 Windows 操作系统丰富的用户界面功能 |
| System.XML            | 包含基于标准的处理 XML 的类                                  |
