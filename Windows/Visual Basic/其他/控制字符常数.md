`ControlChars` 模块包含用作控制字符的常数：

| [Back](https://learn.microsoft.com/zh-cn/dotnet/api/microsoft.visualbasic.controlchars.back?view=net-7.0#microsoft-visualbasic-controlchars-back) | 表示退格符 (`vbBack`)。                   |
| ------------------------------------------------------------ | ----------------------------------------- |
| [Cr](https://learn.microsoft.com/zh-cn/dotnet/api/microsoft.visualbasic.controlchars.cr?view=net-7.0#microsoft-visualbasic-controlchars-cr) | 表示回车符 (`vbCr`)。                     |
| [CrLf](https://learn.microsoft.com/zh-cn/dotnet/api/microsoft.visualbasic.controlchars.crlf?view=net-7.0#microsoft-visualbasic-controlchars-crlf) | 表示回车/换行组合符 (`vbCrLf`)。          |
| [FormFeed](https://learn.microsoft.com/zh-cn/dotnet/api/microsoft.visualbasic.controlchars.formfeed?view=net-7.0#microsoft-visualbasic-controlchars-formfeed) | 表示用于打印功能的换页符 (`vbFormFeed`)。 |
| [Lf](https://learn.microsoft.com/zh-cn/dotnet/api/microsoft.visualbasic.controlchars.lf?view=net-7.0#microsoft-visualbasic-controlchars-lf) | 表示换行符 (`vbLf`)。                     |
| [NewLine](https://learn.microsoft.com/zh-cn/dotnet/api/microsoft.visualbasic.controlchars.newline?view=net-7.0#microsoft-visualbasic-controlchars-newline) | 表示新行字符 (`vbNewLine`)。              |
| [NullChar](https://learn.microsoft.com/zh-cn/dotnet/api/microsoft.visualbasic.controlchars.nullchar?view=net-7.0#microsoft-visualbasic-controlchars-nullchar) | 表示 null 字符 (`vbNullChar`)。           |
| [Quote](https://learn.microsoft.com/zh-cn/dotnet/api/microsoft.visualbasic.controlchars.quote?view=net-7.0#microsoft-visualbasic-controlchars-quote) | 表示双引号字符。                          |
| [Tab](https://learn.microsoft.com/zh-cn/dotnet/api/microsoft.visualbasic.controlchars.tab?view=net-7.0#microsoft-visualbasic-controlchars-tab) | 表示制表符 (`vbTab`)。                    |
| [VerticalTab](https://learn.microsoft.com/zh-cn/dotnet/api/microsoft.visualbasic.controlchars.verticaltab?view=net-7.0#microsoft-visualbasic-controlchars-verticaltab) | 表示垂直制表符 (`vbVerticalTab`)。        |
