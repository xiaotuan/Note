`My.Computer` 对象是 `My` 中的三个中心对象之一，提供对信息和常用功能的访问权限。 `My.Computer` 提供用于访问运行应用程序的计算机的方法、属性和事件。其对象包括：

+ [Audio](https://docs.microsoft.com/zh-cn/dotnet/api/microsoft.visualbasic.devices.audio?view=netframework-4.8)
+ 剪贴板（ [ClipboardProxy](https://docs.microsoft.com/zh-cn/dotnet/api/microsoft.visualbasic.myservices.clipboardproxy?view=netframework-4.8) )
+ [Clock](https://docs.microsoft.com/zh-cn/dotnet/api/microsoft.visualbasic.devices.clock?view=netframework-4.8)
+ [FileSystem](https://docs.microsoft.com/zh-cn/dotnet/api/microsoft.visualbasic.fileio.filesystem?view=netcore-3.1)
+ [Info](https://docs.microsoft.com/zh-cn/dotnet/api/microsoft.visualbasic.devices.servercomputer.info?view=netframework-4.8)
+ [Keyboard](https://docs.microsoft.com/zh-cn/dotnet/api/microsoft.visualbasic.devices.keyboard?view=netframework-4.8)
+ [Mouse](https://docs.microsoft.com/zh-cn/dotnet/api/microsoft.visualbasic.devices.mouse?view=netframework-4.8)
+ [Network](https://docs.microsoft.com/zh-cn/dotnet/api/microsoft.visualbasic.devices.network?view=netframework-4.8)
+ [Ports](https://docs.microsoft.com/zh-cn/dotnet/api/microsoft.visualbasic.devices.ports?view=netframework-4.8)
+ 注册表（ [RegistryProxy](https://docs.microsoft.com/zh-cn/dotnet/api/microsoft.visualbasic.myservices.registryproxy?view=netframework-4.8) ）
