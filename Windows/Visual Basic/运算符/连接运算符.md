连接运算符有两个，即 "+" 和 "&"。其中，"&" 连接运算符用于强制将两个表达式作为字符串连接。而 "+" 连接运算符则不同：当两个表达式都为字符串时，将两个字符串连接；如果一个字符串而另一个是数字，则进行相加，结果为两个数字相加的和。例如：

```vb
a = 2 +3	' a 值为 5
a = "2" + "3" ' a 值为 23
a = "吉林省" & "长春市"	' 吉林省长春市
a = "a1" + 3 ' 出现 "类型不匹配" 的提示信息
a = "a1" & 3	' a 值为 a13
```

> 注意：变量名与 "&" 之间一定要加一个空格。因为 "&" 本身还是长整型的类型符，不加空格容易出现视觉和理解上的误差。
