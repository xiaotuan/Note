逻辑运算符按照运算优先级由高到底排序依次为逻辑非（`Not`）、逻辑与（`And`）、逻辑或（`Or`）、逻辑异或（`Xor`）、逻辑等于（`Eqv`）及逻辑蕴含（`Imp`）。逻辑运算得出的结果是布尔型值，也就是 True 和 False。

| 运算符 | 说明                                                         |
| ------ | ------------------------------------------------------------ |
| And    | 当两边的值皆为 True 时为 True                                |
| Not    | 对表达式结果取反。表达式的值为 False，则返回 True；否则返回 False |
| Or     | 表达式任一边的值为 True 则为 True                            |
| Xor    | 当且仅当表达式的一边为 True 时为 True                        |
