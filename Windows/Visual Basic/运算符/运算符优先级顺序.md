| 算术运算                               | 逻辑运算 |
| -------------------------------------- | -------- |
| 乘方 `^`                               | Not      |

| 乘除（`*`、`/`）                       | Or       |
| 整除（`\`）                            |          |
| 求模（Mod)                             |          |
| 加减（`+`、`-`）                       |          |
| 字符串拼接（`&`）                      |          |
