## 10.4　本章小结

Docker Swarm是使Docker规模化的关键方案。

Docker Swarm的核心包含一个安全集群组件和一个编排组件。

安全集群管理组件是一个企业级的安全套件，提供了丰富的安全机制以及HA特性，这些都是自动配置好的，并且非常易于调整。

编排组件允许用户以一种简单的声明式的方式来部署和管理微服务应用。它不仅支持猿声的Docker Swarm应用，还支持Kubernetes应用。

本书将在第14章对如何使用声明式的方式部署微服务展开更加深入的探讨。
