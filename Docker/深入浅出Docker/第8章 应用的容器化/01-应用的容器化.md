# 第8章　应用的容器化

Docker的核心思想就是如何将应用整合到容器中，并且能在容器中实际运行。

将应用整合到容器中并且运行起来的这个过程，称为“容器化”（Containerizing），有时也叫作“Docker化”（Dockerizing）。

本章将逐步介绍容器化一个简单的Linux Web应用的过程。如果读者没有一个Linux的Docker环境来跟进练习，那么可以免费使用Play With Docker。只需使用浏览器打开Play With Docker的页面，并启动若干Linux Docker节点即可。这是我最喜欢的启动和练习Docker的方式。

本章的内容主要分为3个部分。

+ 简介。
+ 详解。
+ 命令。
