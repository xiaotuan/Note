当我们不再需要访问某个已挂载分区时，可以通过 `umount` 命令将其从挂载点卸除，命令格式如下：

```shell
umount [参数] -t [文件系统类型] [设备名称]
```

主要参数如下：

+ `-a`：卸载 `/etc/mtab` 中的所有文件系统。
+ `-h`：显示帮助。
+ `-n`：卸载时不要将信息存入到 `/etc/mtab` 文件中
+ `-r`：如果无法成功卸载，则尝试以只读的方式重新挂载。
+ `-t <文件系统类型>`：仅卸载选项中指定的文件系统。
+ `-v`：显示执行过程。

例如：

```shell
$ sudo umount -t vfat /dev/sdb1
```
