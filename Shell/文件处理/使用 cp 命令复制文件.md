Linux 下复制文件命令为 `cp`，命令描述如下：

```shell
cp [参数] [源地址] [目的地址]
```

主要参数描述如下：

+ `-a`：此参数和同事指定 `-dpR` 参数相同。
+ `-d`：在复制有符号链接的文件时，保留原始的链接。
+ `-f`：强行复制文件，不管要复制的文件是否已经存在于目标目录。
+ `-I`：覆盖现有文件之前询问用户。
+ `-p`：保留源文件或者目录的属性。
+ `-r` 或 `-R`：递归处理，将指定目录下的文件及子目录一并处理。

例如：

```shell
$ cp a.c b.c
$ cp *.c ../test2
```

通配符 `*` 表示复制所有以 `.c` 结尾的文件。
