### 1. 使用方法

```shell
$ wc GMS/APE_API_KEY/gts-windriver-public.json 
  12   26 2384 GMS/APE_API_KEY/gts-windriver-public.json
```

+ 12 ：行数
+ 26 ：字数
+ 2384 ：Byte 数

### 2. 计算文件的 Byte 数

```shell
$ wc -c gts-windriver-public.json 
2384 gts-windriver-public.json
```

### 3. 计算文件的行数

```shell
$ wc -l gts-windriver-public.json 
12 gts-windriver-public.json
```

### 4. 计算文件最长的行的长度

```shell
Y$ wc -L gts-windriver-public.json 
1756 gts-windriver-public.json
```

### 5. 计算文件的单词数

```shell
$ wc -w gts-windriver-public.json 
26 gts-windriver-public.json
```

### 6. 计算文件的字符数

```shell
$ wc -m gts-windriver-public.json 
2384 gts-windriver-public.json
```
