| 选项               | 含义                               |
| ------------------ | ---------------------------------- |
| -depth             | 在查看目录本身之前先搜索目录的内容 |
| -follow            | 跟随符号链接                       |
| -maxdepths N       | 最多搜索 N 层目录                  |
| -mount（或 -xdev） | 不搜索其他文件系统中的目录         |
