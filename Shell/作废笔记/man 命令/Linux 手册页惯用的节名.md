<center><b>Linux 手册页惯用的节名</b></center>

| 节            | 描述                       |
| ------------- | -------------------------- |
| Name          | 显示命令名和一段简短的描述 |
| Synopsis      | 命令的语法                 |
| Description   | 命令的一般性描述           |
| Options       | 命令选项描述               |
| Exit Status   | 命令的退出状态指示         |
| Return Value  | 命令的返回值               |
| Errors        | 命令的错误消息             |
| Environment   | 描述所使用的环境变量       |
| Files         | 命令用到的文件             |
| Versions      | 命令的版本信息             |
| Conforming To | 命名所遵从的标准           |
| Notes         | 其他有帮助的资料           |
| Bugs          | 提供提交 bug 的途径        |
| Example       | 展示命令的用法             |
| Authors       | 命令开发人员的信息         |
| Copyright     | 命令源代码的版权状况       |
| See Also      | 与该命令类型的其他命令     |
