通过 `sudo` 命令可以使我们暂时将身份切换到 `root` 用户。当使用 `sudo` 命令的时候是需要输入密码的，这里要注意输入密码的时候是没有任何提示的！命令格式如下：

```shell
sudo [选项] [命令]
```

选项主要参数如下：

+ `-h`：显示帮助信息。
+ `-l`：列出当前用户可执行与不可执行的命令。
+ `-p`：改变询问密码的提示符。

```shell
$ adduser
adduser：只有 root 才能将用户或组添加到系统。
```
