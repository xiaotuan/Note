命令 `chown` 用来修改某个文件或者目录的归属者用户或者用户组，命令格式如下：

````shell
chown [参数] [用户名.<组名>] [文件名/目录]
````

其中 `[用户名.<组名>]` 表示要将文件或者目录改为哪一个用户或者用户组，用户名和组名用 `.` 隔开，其中用户名和组名中的任何一个都可以省略，命令主要参数如下：

+ `-c`：效果同 `-v` 类似，但仅显示更改的部分。
+ `-f`：不显示错误信息。
+ `-h`：只对符号链接的文件做修改，不改动其它任何相关的文件。
+ `-R`：递归处理，将指定的目录下的所有文件和子目录一起处理。
+ `-v`：显示处理过程。

例如，我们将文件 test.txt 归属用户更改为 root 用户，所属的用户组也改为  root：

```shell
$ sudo chown root.root test.txt
[sudo] xiaotuan 的密码： 
xiaotuan@xiaotuan:~$ ls -l test.txt
-rw-rw-r-- 1 root root 68 12月  1 19:48 test.txt
```
