| 名称           | 快捷键                                                   | 描述                                                         |
| -------------- | -------------------------------------------------------- | ------------------------------------------------------------ |
| Open Terminal  | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>N</kbd>        | 在新的 GNOME Terminal 窗口中启动一个新的 shell 会话          |
| Open Tab       | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>T</kbd>        | 在现有的 GNOME Terminal 窗口的新标签中启动一个新的 shell 会话 |
| Close Tab      | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>W</kbd>        | 关闭当前标签中的会话                                         |
| Close Window   | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Q</kbd>        | 关闭当前的 GNOME Terminal 会话                               |
| Copy           | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>C</kbd>        | 将所选的文本复制到 GNOME 的剪贴板中                          |
| Paste          | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>V</kbd>        | 将 GNOME 剪贴板中的文本粘贴到会话中                          |
| Full Screen    | <kbd>F11</kbd>                                           | 打开 / 关闭终端窗口全桌面显示模式                            |
| Zoom In        | <kbd>Ctrl</kbd> + <kbd>+</kbd>                           | 逐步增大窗口显示字号                                         |
| Zoom Out       | <kbd>Ctrl</kbd> + <kbd>-</kbd>                           | 逐步减小窗口显示字号                                         |
| Normal Size    | <kbd>Ctrl</kbd> + <kbd>0</kbd>                           | 恢复默认字号                                                 |
| Find           | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>F</kbd>        | 打开 Find 窗口，提供待搜索选项                               |
| Find Next      | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>H</kbd>        | 从终端会话的当前位置开始向前搜索指定文本                     |
| Find Previous  | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>G</kbd>        | 从终端会话的当前位置开始向后搜索指定文本                     |
| Next Tab       | <kbd>Ctrl</kbd> + <kbd>PageDown</kbd>                    | 使下一个标签称为活动标签                                     |
| Previous Tab   | <kbd>Ctrl</kbd> + <kbd>PageUp</kbd>                      | 使上一个标签成为活动标签                                     |
| Move Tab Left  | <kbd>Shift</kbd> + <kbd>Ctrl</kbd> + <kbd>PageUp</kbd>   | 将当前标签移动到前一个标签的前面                             |
| Move Tab Right | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>PageDown</kbd> | 将当前标签移动到下一个标签的后面                             |



+ 

+ 在打开终端后，可以使用快捷键 <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>N</kbd> 在新的 GNOME Terminal 窗口中启动一个新的 `shell` 会话。

+ 在打开终端后，可以使用快捷键 <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>T</kbd> 在现有的 GNOME Terminal 窗口的新标签中启动一个新的 `shell` 会话。
+ 可以使用快捷键 <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>W</kbd> 关闭当前标签中的会话。
