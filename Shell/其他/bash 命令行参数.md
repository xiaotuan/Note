<center><b>bash 命令行参数</b></center>

| 参数      | 描述                                         |
| --------- | -------------------------------------------- |
| -c string | 从 string 中读取命令并进行处理               |
| -i        | 启动一个能够接收用户输入的交互 shell         |
| -l        | 以登录 shell 的形式启动                      |
| -r        | 启动一个受限 shell，用户会被限制在默认目录中 |
| -s        | 从标准输入中读取命令                         |
