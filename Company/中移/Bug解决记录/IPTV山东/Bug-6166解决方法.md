**Bug：**

网络配置》有线网络设置页面，当焦点在PPPoE，遥控器按上键，焦点移到了蓝牙设置

**重现步骤：**

【步骤】

硬件型号：CM201-2
版本号：004.463.110（现网版本）
1、设置-网络配置-有线网络设置页面
2、焦点移到PPPoE
3、遥控器按【上】键

【结果】
3、焦点移到了蓝牙设置

【期望】

**问题分析：**

这个是由于没有对最顶层的焦点设置 `android:nextForceUp` 属性造成焦点由系统自己决定导致的。

**问题解决方法：**

出问题的界面是由 EthNetwork 类实现的，其布局文件是 eth_network.xml 。在 id 为 framenetwork_show_eth_pppoe 的 LinearLayout 节点处添加 `android:nextFocusUp="@+id/framenetwork_show_eth_pppoe"` 代码即可。修改代码如下：

```xml
<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:sw="http://schemas.android.com/apk/res/net.sunniwell.app.swsettings"
    android:id="@+id/network_eth_set_content"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:background="@color/trans"
    android:orientation="vertical" >

    ....

    <LinearLayout
        android:id="@+id/framenetwork_show_eth_pppoe"
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:background="@drawable/selector_setitem"
        android:focusable="true"
        android:focusableInTouchMode="true"
        android:nextFocusLeft="@+id/leftMenu_layout"
        android:nextFocusUp="@+id/framenetwork_show_eth_pppoe"
        android:orientation="vertical"
        android:visibility="visible" >

        <LinearLayout
            android:layout_width="match_parent"
            android:layout_height="wrap_content"
            android:gravity="center_vertical"
            android:nextFocusForward="@+id/radio_eth_pppoe" >

            <RadioButton
                android:id="@+id/radio_eth_pppoe"
                style="@style/net_radio"
                android:layout_width="0dip"
                android:layout_marginLeft="@dimen/setitem_left_margin"
                android:layout_weight="1"
                android:focusable="false"
                android:focusableInTouchMode="false"
                android:nextFocusLeft="@+id/leftMenu_layout"
                android:text="@string/network_pppoe"
                android:textSize="@dimen/setitem_text_size1" />

            <TextView
                android:id="@+id/eth_pppoe_info"
                android:layout_width="0dip"
                android:layout_height="wrap_content"
                android:layout_weight="2"
                android:textColor="@color/white"
                android:textSize="@dimen/setitem_text_size1" />
        </LinearLayout>

        <include layout="@layout/divider" />
    </LinearLayout>
    
    ....
    
</LinearLayout>
```
