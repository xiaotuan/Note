### 7.7　Web Audio API

Web Audio API是一个相对较新的概念，比起<audio>标签，它能够更加直接地和浏览器里正在播放的音频进行交互。也许有一天，它能完全改变人们处理浏览器里音频的方式。然而在写这篇文章的时候，只有少数WebKit浏览器（如Chrome、Android版Chrome，Safari、IOS版Safari）支持网络音频的部分接口，只有Firefox计划支持该接口。Firefox有一套自己的音频接口：音频数据接口（The Audio Data API）。此外，Chrome是目前该接口唯一可以正常工作的浏览器。尽管如此，Web Audio API也会成为标准。因此，本书将简要介绍一下这项新兴技术。
