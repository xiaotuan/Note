<center><font size="5"><b>关于浮动定位的一些理解</b></font></center>

如果多个元素都使用相同的浮动定位，则这些元素会在一行内一次排列，如果超出屏幕宽度后将会换行显示。如果第一个元素高度足够高，则换行后的元素会在第一行第一个元素后面显示。也就是说浮动定位元素只要碰到屏幕边界或浮动元素就会停止浮动。
