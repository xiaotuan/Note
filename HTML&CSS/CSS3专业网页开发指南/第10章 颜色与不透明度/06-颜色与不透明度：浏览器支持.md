### 10.5　颜色与不透明度：浏览器支持

| WebKit | Firefox | Opera | IE |
| :-----  | :-----  | :-----  | :-----  | :-----  | :-----  |
| opacity | 是 | 是 | 是 | 否（IE9有望支持） |
| RGBA值 | 是 | 是 | 是 | 否（IE9有望支持） |
| HSL值 | 是 | 是 | 是 | 否（IE9有望支持） |
| HSLA值 | 是 | 是 | 是 | 否（IE9有望支持） |
| currentColor值 | 是 | 是 | 是 | 否（IE9有望支持） |
| appearance | 是 | 是 | 是 | 否 |
