`catalina.bat` 脚本允许输入命令行参数，`catalina.bat` 的使用方法如下：

| 命令行参数 | 描述                                |
| ---------- | ----------------------------------- |
| start      | 在新的 DOS 窗口启动 `Tomcat` 服务器 |
| run        | 在当前 DOS 窗口启动 `Tomcat` 服务器 |
| debub      | 在跟踪模式下启动 `Tomcat` 服务器    |
| stop       | 关闭 `Tomcat` 服务器                |
