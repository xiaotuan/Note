### 第17章　CSS的未来

本书精挑细选的所有这些属性，目前几乎都在至少一种浏览器上得到实现，并且很可能会随着模块的不断成熟，在不久的将来有更多的实现。但是请务必记住：CSS3在许多方面仍然处于不稳定的状态，一直处在新属性被定义而旧属性被更新的这样一个过程中。

在最后的这一章，我们将了解那些只有部分实现或者根本还没有实现的属性，此外还有已经编写但仍然处于未确定状态的模块，以及还没被批准的新模块提议。尽管我们明年很可能还无法使用这些属性，但几年之后也许就可以使用了。

本章的属性能够让你执行一些数学运算、定义常量和变量，让CSS涉足编程语言的领域。它们借鉴印刷设计的方法去创建布局网格，它们极大地扩展了背景图片以及其他技术的可能性。当然，也别理所当然地认为你会很快就可以在浏览器中看到这些新的属性——如果真的可以看到的话。在这些属性脱离理论阶段之前，还有大量的谈判和辛劳的工作需要去做。如果本章有任何内容刺激到你或者让你觉得愤怒，请公开地让大家知道。W3C会关注网站作者的讨论——特别是如果这些讨论包含了建设性的批评意见的话。

即便如此，我们还是来看一看CSS（可能）的未来将会如何。
