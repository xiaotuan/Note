## 浏览器

四种主要浏览器引擎都拥有一份专用文档和一个资源网站，如果没有的话，这本书的撰写会困难得多。尽管我仍然必须在每个不同的浏览器中测试我的所有例子和演示，才能了解哪个属性会被支持，哪个需要使用前缀，等等，但是这些资源在我最初设计这些例子的时候是非常有用的。

### WebKit

尽管WebKit项目本身并没有太多对你来说有用的文档（同样的不幸的是，这在许多开源项目上这都是太平常不过的事儿），但Apple在它们的Safari开发人员网站上倒是有丰富的文档：

<a class="my_markdown" href="['http://developer.apple.com/safari/']">http://developer.apple.com/safari/</a>

其中与我们关系最为密切的部分就是它们的CSS参考，它会列出所有支持的属性，包括带-webkit前缀的属性：

<a class="my_markdown" href="['http://developer.apple.com/safari/library/documentation/AppleApplications/Reference/<br/%3ESafariCSSRef/']">http://developer.apple.com/safari/library/documentation/AppleApplications/Reference/
SafariCSSRef/</a>

### Firefox

Firefox倒是不符合我刚刚提到的开源文档的趋势，在它们那杰出的Mozilla开发人员网络（Mozilla Developer Network）上有极为出色的CSS参考。该站点详细解释了所有的属性，并带有例子和图示，在我写这本书的时候，在许许多多的地方都参考了它：

<a class="my_markdown" href="['https://developer.mozilla.org/en/CSS_Reference/']">https://developer.mozilla.org/en/CSS_Reference/</a>

它们也在一个单独的页面上列出了所有-moz-前缀的属性：

<a class="my_markdown" href="['https://developer.mozilla.org/en/CSS_Reference/Mozilla_Extensions/']">https://developer.mozilla.org/en/CSS_Reference/Mozilla_Extensions/</a>

### Opera

Opera使用的布局引擎称为Presto，最新的版本是2.7，装备在Opera11上。Opera的文档页面很有用地列出Presto支持的所有CSS，其中有一些全面详尽的表格，精确地列出了各个模块的实现程度：

<a class="my_markdown" href="['http://www.opera.com/docs/specs/presto27/']">http://www.opera.com/docs/specs/presto27/</a>

### Internet Explorer

微软开发者网络（Microsoft Developer Network）有一个叫做“CSS兼容性与Internet Explorer”（CSS Compatibility and Internet Explorer）的页面，列出了微软浏览器的许多版本中所有实现的CSS特性：

<a class="my_markdown" href="['http://msdn.microsoft.com/en-us/library/cc351024(VS.85']">http://msdn.microsoft.com/en-us/library/cc351024(VS.85).aspx</a>.aspx)

如果你读到这里的时候，IE9还没有发布，那么可以看看IE9的开发人员指南，其中包含了关于即将到来的CSS3支持的大量信息：

<a class="my_markdown" href="['http://msdn.microsoft.com/en-gb/ie/ff468705.aspx']">http://msdn.microsoft.com/en-gb/ie/ff468705.aspx</a>
