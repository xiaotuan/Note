## 代码生成工具

如果手上没有这本书的话，要记住所有带前缀的属性在实现上的小差异，可能是相当困难的，手动键入所有的属性也可能是一件繁重的任务。有几个网站意识到了这个问题，创建出一些工具，使得我们在编写跨浏览器的代码时不用再干这种单调无聊的工作。

### CSS3, Please!

许多特性都需要使用不同的带前缀的属性，CSS3, Please!不仅可以帮助你处理重复输入这些属性的问题，还提供了这些特性的实时预览。在任何属性中更新一个值，都会自动地更新它的所有兄弟属性。所以当你对它的输出觉得满意的时候，可以复制并粘贴代码。CSS3, Please!还往前迈出一步，复制了一些带专用的微软滤镜的CSS3特性，但如果我是你的话，我会忽略掉这些功能，因为它们是不标准的。以下是该网站的URL：

<a class="my_markdown" href="['http://www.css3please.com']">http://www.css3please.com</a>

### CSS3 Generator

CSS3 Generator实现了类似的功能，但是多了几个属性，在界面上也有很大差异。它也提供了微软专用的滤镜，只允许使用以像素为长度的值，但这个工具是一个有用的小帮手：

<a class="my_markdown" href="['http://www.css3generator.com/']">http://www.css3generator.com/</a>

### CSS3 Gradient Generator

回顾第11章，我讨论了CSS渐变，并且花了很长时间解释Firefox和WebKit语法之间的区别。CSS3 Gradient Generator帮助我们处理创建渐变时所有繁重的工作和计算，并且可以生成符合规范的跨浏览器代码，在大多数情况下应该可以帮助你排忧解难。请查看：

<a class="my_markdown" href="['http://gradients.glrzad.com/']">http://gradients.glrzad.com/</a>

### Type Folly

Type Folly是一个仅仅为了试验排版特性而构建的工具。Photoshop用户应该马上就能熟悉它的界面，它允许你创建并编辑文本层，为每一层添加字体、文本和2D变换属性。该工具对于实现我希望的效果还需要一些技巧，但我相信这需要练习，你会发现它对于排版实验是很有用的：

<a class="my_markdown" href="['http://www.typefolly.com']">http://www.typefolly.com</a>
