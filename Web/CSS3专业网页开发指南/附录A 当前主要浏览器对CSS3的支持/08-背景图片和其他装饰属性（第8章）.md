## 背景图片和其他装饰属性（第8章）

| **WebKit** | **Firefox** | **Opera** | **IE** |
| :-----  | :-----  | :-----  | :-----  | :-----  | :-----  |
| 多背景图片 | 是 | 是 | 是 | 否（IE9有望支持） |
| background-size | 是（在Safari 4及较早版本中带前缀） | 是（带前缀；Firefox 4有望不带前缀） | 是 | 否（IE9有望支持） |
| background-clip | 是（在Safari 4及较早版本中带前缀） | 是（带前缀；Firefox 4有望不带前缀） | 是 | 否（IE9有望支持） |
| background-origin | 是（在Safari 4及较早版本中带前缀） | 是（带前缀；Firefox 4有望不带前缀） | 是 | 否（IE9有望支持） |
| background-repeat（新值） | 否 | 否 | 是 | 否（IE9有望支持） |
| image-rect | 否 | 是（带前缀） | 否 | 否 |
| mask-* | 是（带前缀） | 否 | 否 | 否 |
