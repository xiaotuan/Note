### 1. GIF 动画

**案例：示例 13-01：在页面中使用 GIF 动画**

```html
<!doctype html>
<html>
    <head>
        <meta charset="utf-8">
        <meta keywords="HTML5 CSS GIF">
        <meta content="在页面中使用GIF动画">
        <title>在页面中使用GIF动画</title>
    </head>
    <style>
        .show-box .box {
            width: 150px;
            height: 100px;
            float: left;
        }
    </style>
    </head>

    <body>
        <div class="show-box">
            <div class="box"><img src="images/13-1/2.gif" alt="" /></div>
            <div class="box"><img src="images/13-1/3.gif" alt="" /></div>
            <div class="box"><img src="images/13-1/4.gif" alt="" /></div>
            <div class="box"><img src="images/13-1/5.gif" alt="" /></div>
            <div class="box"><img src="images/13-1/6.gif" alt="" /></div>
        </div>
    </body>
</html>
```
