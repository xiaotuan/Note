两个数字常用的函数是 `round()` 和 `number_format()`。前者用于把小数四舍五入为最近的整数：

```php
$n = 3.14;
$n = round($n);	// 3
```

或者把小数四舍五入到指定的位数：

```php
$n = 3.142857;
$n = round($n, 3);	// 3.143
```

`number_format()` 函数用于把一个数字转换成更常见的表示形式，用逗号作为千位分隔符，例如：

```php
$n = 20943;
$n = number_format($n);	// 20,943
```

这个函数还可以设置小数点的指定位数：

```php
$n = 20943;
$n = number_format($n, 2);	// 20,943.00
```
