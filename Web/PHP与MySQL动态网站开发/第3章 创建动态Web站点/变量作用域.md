因为包含文件像它们是原始（包含）脚本的一部分那样工作，所以在 `include()` 那一行之前定义的变量可供包含文件使用。

在一个函数外部定义的变量不能在其内部使用。

要改变一个函数内的变量的作用域，可以使用 `global` 语句。

```php
function function_name() {
    global $var;
}

$var = 20;
function_name();	// Function call.
```

避开变量作用域的另一个方法是利用超全局变量： $\_GET、$\_POST、$\_REQUEST等。这些变量在你的函数内时自动可以访问的。也可以添加元素到 $GLOBALS 数组中，使得可以在函数内使用它们。

也就是说，最好不要在函数内使用全局变量。在设计函数时，应该使它们根据需要接受每个值作为参数，并根据需要返回任何值。
