### 7.2.1　组件路由

我们不会用新东西来实现应用的路由。相反，我们将使用 `children` 这个特殊的组件属性。也许还记得前面章节中的 `children` 属性，它是 `React.createElement(type, props, children)` 方法签名的一部分或者作为用来组合组件的特殊属性。

之前仅从输入视角关注了children：将一些组件传递给另一个组件来把它们组合到一起。现在从组件内部访问 `children` 并利用组件自身来搭建路由。这就是开始将组件映射到URL的地方。如果Web开发中的路由是将URL映射到行为或视图，那么React中的路由就是将URL映射到特定的组件。
