当在浏览器中打开spa.html时，看到的滑块是收起的，如图1-5所示。我们使用了流式布局（liquid layout），界面按显示大小自适应，滑块总是固定在右下角。我们没有给容器添加边框，因为这会增加容器的宽度，从而妨碍开发，因为必须修改容器的大小来适应这些边框。在创建并验证了基本布局之后再来添加边框是很方便的，之后的章节就是这么做的。
