

#### 
  3.6.1 理解浏览器用户所期望的行为


在要维护哪方面的信息上，桌面应用和Web应用的差别很大。如果桌面应用不提供“回退”的功能，它就可以省去“上一步”的按钮。但是在Web应用中，浏览器的回退按钮（最常使用的浏览器控件之一，当面盯着用户，乞求被用户点击），我们无法移除它。

前进按钮、标签按钮和查看历史也一样。用户期望这些历史控件能够工作。如果不能，用户就会很恼火，我们的应用就绝不会在网站中脱颖而出。表3-1展示了和这些历史控件对应的类似桌面控件。
