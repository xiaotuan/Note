建议你把第4章的整个目录结构复制到一个新的“chapter_5”目录里面，这样就可以在新目录里面更新文件了。我们来回顾一下第4章留下的文件结构，如代码清单5-1所示。

下面是我们计划要做的修改。

创建Avatar的CSS样式表。

修改Shell的CSS样式表，支持用户登入。

引入统一的触摸和鼠标输入的jQuery插件。
