### 14.4.2　 `switchClass ()` 方法

除了 `addClass ()` 、 `removeClass ()` 和 `toggleClass ()` 方法有所改进之外，jQuery UI中还包含了一个全新的 `switchClass ()` 方法，用来把元素从一个CSS类切换到另一个CSS类。

`switchClass ()` 方法的用法如下：

```css
$(selector, context). switchClass (classNameRemoved, classNameAdded, duration,
　　　　　　　　　　　　　　　　　　　　　easing, callback);
```
