### 14.1.4　 `shake` 特效

`shake` 特效能使元素以水平或垂直地摇晃的方式出现或消失。表14-4中列出了此特效的相关选项。

<center class="my_markdown"><b class="my_markdown">表14-4　管理 `shake` 特效的选项</b></center>

| 选项 | 功能 |
| :-----  | :-----  | :-----  | :-----  |
| `options.direction` | 指定元素摇晃的方向： `"up"` （上）、  `"down"` （下）、  `"left"` （左，也是默认值）或"right"（右） |
| `options.distance` | 每一次摇晃的距离（单位为像素）。默认值为20个像素 |
| `options.times` | 摇晃的次数。默认值为3次 |
