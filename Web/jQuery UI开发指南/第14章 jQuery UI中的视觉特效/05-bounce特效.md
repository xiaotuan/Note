### 14.1.3　 `bounce` 特效

`bounce` 特效能使元素以水平或垂直地弹跳的方式出现或消失。表14-3中列出了此特效的相关选项。

<center class="my_markdown"><b class="my_markdown">表14-3　管理 `bounce` 特效的选项</b></center>

| 选项 | 功能 |
| :-----  | :-----  | :-----  | :-----  |
| `options.mode` | 显示（ `"show"` ）或隐藏（ `"hide"` ）元素。默认值为 `"effect"` ，此时只会制造弹跳效果，并不会伴随元素的出现或消失 |
| `options.direction` | 指定元素弹跳的方向： `"up"` （上，也是默认值）、  `"down"` （下）、  `"left"` （左）或 `"right"` （右） |
| `options.distance` | 每一次弹跳的距离（单位为像素）。默认值为20个像素 |
| `options.times` | 弹跳的次数。默认值为5次 |
