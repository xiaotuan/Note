### 10.5　在保管元素上使用 `bind ()` 管理事件

除了 `droppable (options)` 中的 `options` 参数提供的事件方法外，jQuery UI还允许我们使用 `bind ()` 方法来管理这些事件（详见表10-16）。

<center class="my_markdown"><b class="my_markdown">表10-16　jQuery UI创建的事件</b></center>

| 事件 | 功能 |
| :-----  | :-----  | :-----  | :-----  |
| `dropactivate` | 等同于 `options.activate` |
| `dropdeactivate` | 等同于 `options.deactivate` |
| `dropover` | 等同于 `options.over` |
| `dropout` | 等同于 `options.out` |
| `drop` | 等同于 `options.drop` |
