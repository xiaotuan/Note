### 12.4　使用 `bind()` 管理元素调换事件

jQuery UI允许使用 `bind()` 方法来管理元素调换事件，详见表12-13。

<center class="my_markdown"><b class="my_markdown">表12-13　jQuery UI创建的事件</b></center>

| 事件 | 功能 |
| :-----  | :-----  | :-----  | :-----  |
| `sortstart` | 等同于 `options.start` |
| `sortstop` | 等同于 `options.stop` |
| `sortbeforestop` | 等同于 `options.beforeStop` |
| `sort` | 等同于 `options.sort` |
| `sortchange` | 等同于 `options.change` |
| `sortupdate` | 等同于 `options.update` |
| `sortreceive` | 等同于 `options.receive` |
| `sortremove` | 等同于 `options.remove` |
| `sortactivate` | 等同于 `options.activate` |
| `sortdeactivate` | 等同于 `options.deactivate` |
