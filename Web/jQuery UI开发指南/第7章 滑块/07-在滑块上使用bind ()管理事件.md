### 7.4　在滑块上使用 `bind ()` 管理事件

除了在 `slider (options)` 方法的选项中使用事件方法外，jQuery UI还允许使用 `bind ()` 方法来管理这些事件（如表7-5描述的）。

<center class="my_markdown"><b class="my_markdown">表7-5　jQuery UI创建的事件种类</b></center>

| 事件 | 功能 |
| :-----  | :-----  | :-----  | :-----  |
| `slidestart` | 等同于 `options.start` |
| `Slidestop` | 等同于 `options.stop` |
| `slidechange` | 等同于 `options.change` |
| `slide` | 等同于 `options.slide` |
