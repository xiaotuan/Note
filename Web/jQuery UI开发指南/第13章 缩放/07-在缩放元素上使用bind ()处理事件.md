### 13.4　在缩放元素上使用 `bind ()` 处理事件

除了 `resizable (options)` 中的 `options` 参数提供的事件方法外，jQuery UI还允许使用 `bind ()` 方法来管理这些事件（详见表13-9）。

<center class="my_markdown"><b class="my_markdown">表13-9　 `jQuery UI` 创建的事件</b></center>

| 事件 | 功能 |
| :-----  | :-----  | :-----  | :-----  |
| `resizestart` | 等同于 `options.start` |
| `resizestop` | 等同于 `options.stop` |
| `resize` | 等同于 `options.resize` |
