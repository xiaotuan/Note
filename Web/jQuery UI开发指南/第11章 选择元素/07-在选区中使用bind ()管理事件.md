### 11.4　在选区中使用 `bind ()` 管理事件

除了 `selectable (options)` 中的 `options` 参数提供的事件方法外，jQuery UI还允许使用 `bind()` 方法来管理这些事件。表11-4中列出了这些事件。

<center class="my_markdown"><b class="my_markdown">表11-4　jQuery UI创建的事件</b></center>

| 事件 | 功能 |
| :-----  | :-----  | :-----  | :-----  |
| `selectablestart` | 等同于 `options.start` |
| `Selectablestop` | 等同于 `options.stop` |
| `selectbaleselecting` | 等同于 `options.selecting` |
| `selectableunselecting` | 等同于 `options.unselecting` |
| `selectableselected` | 等同于 `options.selected` |
| `selectableunselected` | 等同于 `options.unselected` |
