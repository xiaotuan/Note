在网页中可以通过 `strike` 参数对文字添加删除线效果。

### 1. 语法

```html
<strike>文字</strike>
或
<s>文字</s>
```

### 2. 示例代码

```html
<html>
    <head>
    	<title>文字的删除线效果</title>
    </head>
    <body>
        正常的文字效果<br />
        在文字上使用s 标记来添加删除线<br />
        <s>删除文字的效果</s><br/><br/>
        在文字上使用strike 标记来添加删除线<br />
        <strike>删除文字的效果</strike>
    </body>
</html>
```
