在网页中可以通过添加对文字的标注来说明网页中的某段文字。

### 1. 语法

```html
<ruby>
    被说明的文字
    <rt>
        文字的标注
    </rt>
</ruby>
```

### 2. 示例代码

```html
 <html>
    <head>
    	<title>添加文字标注</title>
    </head>
    <body>
        <ruby>有情芍药含春泪,无力蔷薇卧晓枝。<br /><br />
        <rt>
        作者秦观
        </rt>
        </ruby>
    </body>
</html>
```
