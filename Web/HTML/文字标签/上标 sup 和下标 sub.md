### 1. 语法

```html
<sup>上标标记</sup>
<sub>下标标记</sub>
```

### 2. 示例代码

```html
<html>
    <head>
   		<title>上标与下标的效果</title>
    </head>
    <body>
        在方程式中应用上标的效果<br/>
        X<sup>3</sup>+9X<sup>2</sup>-3=0<br/><br/>
        在文字中应用下标的效果<br/>
        3X<sub>1</sub>+2X<sub>2</sub>=10
    </body>
</html>
```
