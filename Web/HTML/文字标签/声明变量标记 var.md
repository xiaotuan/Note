而在 HTML 中也提供了一种标记，用于专门设置变量的效果。

### 1. 语法

```html
<var>变量</var>
```

### 2. 示例代码

```html
<html>
    <head>
    	<title>声明变量标记</title>
    </head>
    <body>
        <p>所谓的定义变量就是给变量赋值。</p>
        定义变量的格式为：<br>
        变量名：数值或者表达式的值。<br>
        其中符号“:=”是定义符，又称赋值符。<br>
        <p>例如定义变量<var>x</var>的值为<var>y+6</var>,可以表示为：</p>
        <p><var>x</var>:=<var>y</var>+6
    </body>
</html>
```
