### 1. 设置行的高度——height

#### 1.1 语法

```html
<tr hegith="表格中某行的高度"></tr>
```

#### 1.2 示例代码

```html
<!DOCTYPE html>
<html>
    <head>
        <title>设置行高度</title>
    </head>
    <body>
        <table border="1" bordercolor="#FF0000" cellpadding="5">
            <caption>我国著名诗人——李白</caption>
            <tr height="100">
            	<td>诗名</td>
                <td>登金陵凤凰台</td>
            </tr>
            <tr>
            	<td>内容</td>
                <td>
                	凤凰台上凤凰游，凤去台空江自流。
                   	吴宫花草埋幽径，晋代衣冠成古丘。
                    三台半落青山外，
                </td>
            </tr>
        </table>
    </body>
</html>
```
