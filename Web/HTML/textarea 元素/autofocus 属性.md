HTML5 为 `textarea` 元素增加 `autofocus` 属性，该属性表示在打开页面时使元素自动获得焦点。例如：

```html
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8"> 
        <title>W3Cschool(w3cschool.cn)</title> 
    </head>
    
    <body>

    <textarea rows="4" cols="50" autofocus>
    在 w3cschool.cn 你将学会如何制作网站。我们提供了最全的web技术教程。
    </textarea>

    <p><strong>注意：</strong> Internet Explorer 9 及之前的版本支持 textarea 标签的 autofocus 属性。 </p>

    </body>
</html>
```
