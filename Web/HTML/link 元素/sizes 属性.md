> 提示：
>
> 目前几乎没有主流浏览器支持 sizes 属性。

HTML 5 为 `link` 元素新增 `sizes` 属性，该属性可以指定关联图标（icon）的大小，所以通常与 icon 元素一起使用。例如：

```html
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8"> 
        <title>W3Cschool(w3cschool.cn)</title> 
        <link rel="icon" href="/statics/demosource/demo_icon.gif" type="image/gif" sizes="16x16">
    </head>

    <body>
        <h2>Hello world!</h2>
        <p>在<a href="/statics/demosource/tryhtml5_link_sizes.html" target="_blank">新窗口</a>中打开这个页面看到这个图标。</p>
        <p><b>注意:</b> 目前几乎没有主流浏览器支持 sizes 属性。</p>

    </body>
</html>
```
