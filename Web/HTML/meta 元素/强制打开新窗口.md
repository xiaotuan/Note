强制网页在当前窗口中以独立的页面显示，可以放置自己网页被别人当作一个 frame 页调用。

**语法**

```html
<meta http-equiv="windows-target" content="_top" />
```

> 注意
>
> `windows-target` 表示新网页的打开方式。
>
> content 中设置 `_top` 则代表打开的是一个独立页面。

**示例**

```html
<!DOCTYPE html>
<html>
    <head>
        <title>强制打开新窗口</title>
        <meta http-equiv="windows-target" content="_top" />
    </head>
    <body>
        
    </body>
</html>
```
