`<meta>` 标记用于定义页面信息的名称、关键字、作者等。

`<meta>` 标记不需要设置结束标记。

HTML 头页面中可以有多个 `<meta>` 标记。

`<meta>` 标记的属性有两种：`name` 和 `http-equiv`，其中 `name` 属性主要用于描述网页，以便于搜索引擎机器人查找和分类。

例如：

```html
<!DOCTYPE html>
<html>
    <head>
        <title>Meta 标记的使用</title>
        <!--	设置关键字	-->
        <meta name="keyword" content="html,元信息,关键字"/>
        <!--	设置页面描述	-->
        <meta name="description" content="关于 HTML 使用的网站"/>
        <!--	设置编辑页面所使用的工具	-->
        <meta name="generator" content="Adobe Dreamweaver CC"/>
    </head>
    <body>
        
    </body>
</html>
```
