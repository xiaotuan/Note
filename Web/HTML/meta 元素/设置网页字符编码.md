### 1. 设置网页字符编码

**语法**

```html
<meta http-equiv="Content-Type" content="text/html;charset=utf-8" />
```

**示例**

```html
<!DOCTYPE html>
<html>
    <head>
        <title>设置网页字符编码</title>
        <meta http-equiv="Content-Type" content="text/html;charset=utf-8" />
    </head>
    <body>
        
    </body>
</html>
```

### 2. 设置网页语言

**语法**

```html
<meta http-equiv="Content-Language" content="语言" />
```

**示例**

```html
<!DOCTYPE html>
<html>
    <head>
        <title>设置网页语言</title>
        <meta http-equiv="Content-Language" content="zh_CN" />
    </head>
    <body>
        
    </body>
</html>
```
