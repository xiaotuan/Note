段落与段落之间是隔行换行的，文字的行间距过大，这时可以使用换行标记来完成文字的紧凑换行显示。

### 1. 语法

```html
<br/>
```

一个  `<br>` 标记代表一个换行，连续的多个标记可以多次换行。

### 2. 示例代码

```html
<html>
    <head>
   		<title>文字的换行</title>
    </head>
    <body>
        龚自珍<br/><br/>
        九州生气恃风雷,万马齐喑究可哀。<br/>
        我劝天公重抖擞, 不拘一格降人才。<br/>
    </body>
</html>
```
