使用网页缓存可以加快浏览网页的速度，因为缓存将曾经浏览过的页面暂存在电脑中。但是如果网页的内容频繁地更新，网页制作者希望用户随时都能查看到最新的网页内容，则可以通过 meta 语句禁用页面缓存。

**语法**

```html
<meta http-equiv="cache-control" content="no-cache" />
或
<meta http-equiv="pragma" content="no-cache" />
```

**示例**

```html
<!DOCTYPE html>
<html>
    <head>
        <title>禁止从缓存中调用</title>
        <meta http-equiv="cache-control" content="no-cache" />
        <meta http-equiv="pragma" content="no-cache" />
    </head>
    <body>
        
    </body>
</html>
```
