无序列表使用 `<ul>` 标签表示，标签内包含 `<li>` 标签，其格式如下：

```html
<ul type="列表类型">
    <li>列表内容1</li>
    <li>列表内容2</li>
    <li>列表内容3</li>
</ul>
```

无序列表的 `type` 属性可以使用的值有：

+ `circle`：空心圆
+ `disc`（默认）：实心圆
+ `square`：实心方块

```html
<!DOCTYPE html>
<html>
    <head>
        <title>有序列表标签</title>
    </head>
    <body>
        <ul type="circle">
            <li>财务部</li>
            <li>学工部</li>
            <li>人事部</li>
        </ul>
    </body>
</html>
```

运行效果如下：

<ul type="circle">
    <li>财务部</li>
    <li>学工部</li>
    <li>人事部</li>
</ul>
