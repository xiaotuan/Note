有序列表使用 `<ol>` 标签表示，标签里包含 `<li>` 标签，其格式如下：

```html
<ol type="排序方式">
    <li>列表内容1</li>
    <li>列表内容2</li>
    <li>列表内容3</li>
</ol>
```

有序列表的 `type` 属性值有：

+ `1`（默认）：序列号为 1, 2, 3, 4, 5, 6....
+ `a`：序列号为 a, b, c, d, e, f...
+ `i`：序列号为 i, ii, iii, iv, v...

```html
<!DOCTYPE html>
<html>
    <head>
        <title>有序列表标签</title>
    </head>
    <body>
        <ol type="i">
            <li>财务部</li>
            <li>学工部</li>
            <li>人事部</li>
        </ol>
    </body>
</html>
```

运行效果如下：

<ol type="i">
    <li>财务部</li>
    <li>学工部</li>
    <li>人事部</li>
</ol>
