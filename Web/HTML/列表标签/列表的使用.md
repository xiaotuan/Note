列表标签使用 `<dl>` 标签里面包含 `<dt>` 或 `<dl>` 标签实现，其格式如下：

```html
<dl>
    <dt>列表头</dt>
    <dd>列表内容1</dd>
    <dd>列表内容2</dd>
    <dd>列表内容3</dd>
</dl>
```

```html
<!DOCTYPE html>
<html>
    <head>
        <title>列表标签</title>
    </head>
    <body>
        <dl>
            <dt>传智播客</dt>
            <dd>财务部</dd>
            <dd>学工部</dd>
            <dd>人事部</dd>
        </dl>
    </body>
</html>
```

运行效果如下所示：

<dl>
    <dt>传智播客</dt>
    <dd>财务部</dd>
    <dd>学工部</dd>
    <dd>人事部</dd>
</dl>
