菜单列表主要用于设计单列的菜单列表。菜单列表在浏览器中的显示效果和无序列表是相同的，因此它的功能也可以通过无序列表来实现。

### 1. 语法

```html
<menu>
    <li>列表项1</li>
    <li>列表项2</li>
    <li>列表项3</li>
    ......
</menu>
```

### 2. 示例代码

```html
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title>创建菜单列表</title>
    </head>
    <body>
        <font size="+3" color="#3300FF">本章中介绍的列表主要包括：</font><br />
        <menu>
            <li>无序列表</li>
            <li>有序列表</li>
            <li>定义列表</li>
            <li>菜单列表</li>
            <li>目录列表</li>
        </menu>
    </body>
</html>
```
