在创建列表时，可以单独设置列表中文字的颜色。这里可以直接对文字进行颜色设置。

### 1. 语法

```html
<li><font color="颜色代码">列表项</font></li>
```

> 提示：也可以在列表中进行整体颜色的设置。

### 2. 示例代码

```html
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title>设置列表文字的颜色</title>
    </head>
    <body>
        <font size="+5" color="#666666">金庸名著</font>
        <ul><font color="#0066CC">
            <li>雪山飞狐</li>
            <li>天龙八部</li>
            <li>射雕英雄传</li>
            <li>倚天屠龙记</li>
        </font>
        </ul>
        <hr color="#00FFCC" size="3"/>
        <font size="+5" color="#666666">琼瑶小说</font>
        <ul>
            <li><font color="#9966FF">青青河边草</font></li>
            <li><font color="#9966FF">梅花三弄</font></li>
            <li><font color="#9966FF">烟雨蒙蒙</font></li>
            <li><font color="#9966FF">还珠格格</font></li>
        </ul>
    </body>
</html>
```
