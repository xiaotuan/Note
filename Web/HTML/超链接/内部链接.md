所谓内部链接，指的是在同一个网站内部，不同的 HTML 页面之间的链接关系。在建立网站内部链接的时候，要考虑到使链接具有清晰的导航结构，使用户方便地找到所需要内容的 HTML 文件。

### 1. 示例代码

```html
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title>内部链接的实现</title>
    </head>
    <body>
        <center><h2>中国古城介绍</h2></center>
        <hr size="2" color="#FF0000" />
        <font size="+4">西安古城</font><br /><br />
        世界历史名城，华夏精神故乡，中国第一古都—西安。<br /><br />
        <a href="film/5-3-1.html">详细介绍</a><br />
        <hr size="2" color="#FF0000" />
        <font size="+4">大理古城</font><br /><br /><br />
         大理历史深远，素有“文献名邦”美名。大理古城位于风光绮丽的苍山之麓，始建于明洪武十五年，是中国二十四个历史文化名城之一。<br /><br />
         <a href="film/5-3-2.html">详细介绍</a>
    </body>
</html>
```
