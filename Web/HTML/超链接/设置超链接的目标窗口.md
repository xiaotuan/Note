在创建网页的过程中，有时候并不希望超链接的目标窗口将原来的窗口覆盖，如在打开新的窗口时，主页面的窗口仍保留在原处，这时可以通过 `target` 参数设置目标窗口的属性。

### 1. 语法

```html
<a href="链接地址" target="目标窗口的打开方式">链接元素</a>
```

`target` 参数的取值有 4 种，如下表所示：

| target 值 | 目标窗口的打开方式                             |
| --------- | ---------------------------------------------- |
| _parent   | 在上一级窗口打开，常在分帧的框架页面中使用     |
| blank     | 新建一个窗口打开                               |
| self      | 在同一窗口打开，与默认设置相同                 |
| top       | 在浏览器的整个窗口打开，将会忽略所有的框架结构 |

### 2. 示例代码

```html
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title>目录窗口的打开方式</title>
    </head>
    <body>
        <center><h2>鲁迅作品介绍</h2></center>
        <hr size="4" color="#FF0000" />
        <font size="+4">《呐喊》</font><br /><br />

    </body>
</html>
```
