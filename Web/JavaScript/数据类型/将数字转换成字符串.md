### 1. 将数字转换成不同进制字符串

Number 类定义的 toString() 方法可以接收表示转换基数的可选参数，如果不指定此参数，转换规则将是基于十进制。  

```js
var n = 17;
binary_string = n.toString(2); // 转换为 "10001"
octal_string = "0" + n.toString(8); // 转换为 "021"
hex_string = "0x" + n.toString(16); // 转换为 "0x11"
```

### 2. 将数字转换成指定小数点后位数的字符串

toFixed（）根据小数点后的指定位数将数字转换为字符串，它从不使用指数记数法，可以通过传递参数指定小数点后的位数。  

```js
var n = 123456.789;
n.toFixed(0); // "123457"
n.toFixed(2); // "123456.79"
n.toFixed(5); // "123456.78900"
```

### 3. 将数字转换成指数记数法形式字符串

可以使用 Number 类的 `toExponential()` 方法将数字转换成指数记数法形式的字符串，其中，小数点前只有一位，小数点后的位数则有参数指定。例如：

```js
var n = 123456.789;
n.toExponential(1); // "1.2e+5"
n.toExponential(3); // "1.235e+5"
```

### 4. 根据数字选择合适的字符串形式

`toPrecision()` 根据指定的有效数字位数将数字转换成字符串。如果有效数字的位数少于数字整数部分的位数，则转换成指数形式。

```js
var n = 123456.789;
n.toPrecision(4); // "1.235e+5"
n.toPrecision(7); // "123456.8"
n.toPrecision(10); // "123456.7890"
```
