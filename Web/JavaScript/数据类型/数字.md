和其他变成语言不同，JavaScript 不区分整数值和浮点数值。JavaScript 中的所有数字均用浮点数值表示。

JavaScript采用 IEEE 754 标准定义的 64 位浮点格式表示数字，这意味着它能表示的最大值是 ±1.7976931348623157 x 10<sup>308</sup>，最小值是 ±5 x 10<sup>-324</sup>。

按照 JavaScript 中的数字格式，能够表示的整数范围是从 -9007199254740992 ~ 9007199254740992（即 -2<sup>53</sup> ~ 2<sup>53</sup>），包含边界值。如果使用了超过此范围的整数，则无法保证低位数字的精度。然而需要注意的是，JavaScript 中实际的操作则是基于 32 为整数的。
