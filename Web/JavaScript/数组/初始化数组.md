定义数组的字面形式是在中括号内用逗号区分的任意数量的值，例如：

```js
var colors = [ "red", "blue", "green" ];
console.log(colors[0]);	// "red"
```

上面的代码等价于下面代码：

```js
var colors = new Array("red", "blue", "green");
console.log(colors[0]);	// "red"
```

初始化多维数组：

```js
var matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]];
```

初始化指定位置元素，其他元素默认设置为 undefined。例如：

```js
var sparseArray = [1, , , , , 5];
```
