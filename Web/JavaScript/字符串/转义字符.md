<center><b>JavaScript 转义字符</b></center>

| 转义字符 | 含义                                       |
| -------- | ------------------------------------------ |
| \0       | NULL字符(\u0000）                          |
| \b       | 退格符（\u0008）                           |
| \t       | 水平制表符（\u0009）                       |
| \n       | 换行符（\u000A）                           |
| \v       | 垂直制表符（\u000B）                       |
| \f       | 换页符（\u000C）                           |
| \r       | 回车符（\u000D）                           |
| `\"`     | 双引号（\u0022）                           |
| `\'`     | 撇号或单引号（\u0027）                     |
| `\\`     | 反斜线（\u005C）                           |
| \xXX     | 由两位十六进制数 XX 指定的 Latin-1 字符    |
| \uXXXX   | 由 4 位十六进制数 XXXX 指定的 Unicode 字符 |
