### 1. if 语句

Swift 中的 if 语句的功能与其他大部分语言中的类似。

```swift
let random = arc4random_uniform(10)
if random < 5 {
    print("Hi")
} else {
    print("Ho")
}
```

arc4random_uniform() 函数返回的是 0 (包括0) 与其参数（不包括）之间的整型值。注意，Swift 中把条件判断放在括号中并不是必需的，因此这样的代码也能够运行：

```swift
if random < 5 { // 括号不是必需的
}
if (random < 5) {   // 不过也可以这样使用   
}
```
