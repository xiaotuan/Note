<center><font size="5"><b>算术运算符</b></font></center>

Swift 对所有的数字类型支持四种标准算术运算符：

+ 加 ( + )
+ 减 ( - )
+ 乘 ( * )
+ 除 ( / )

```swift
1 + 2   // equals 3
5 - 3   // equals 2
2 * 3   // equals 6
10.0 / 2.5  // equals 4.0
```

与 C 和 Objective-C 中的算术运算符不同，Swift 算术运算符默认不允许值溢出。你可以选择使用 Swift 的溢出操作符（比如 `a&+b` ） 来行使溢出行为。

加法运算符同时也支持 String 的拼接：

```swift
"hello, " + "world" // equals "hello, world"
```
