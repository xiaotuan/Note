<center><font size="5"><b>组合赋值符号</b></font></center>

Swift 提供了由赋值符号（ = ）和其他符号组成的 组合赋值符号 。一个加赋值符
号的栗子 （ += ）：

```swift
var a = 1
a += 2
// a is now equal to 3
```

表达式 `a += 2` 其实就是 `a = a + 2` 的简写。效率上来讲，加号和赋值符号组合成的一个运算符能够同时进行这两个操作。

> 注意
> 组合运算符不会返回任何值。举例来说，你不能写成这样 `let b = a += 2` 。这个与前边提到的增量和减量符号的行为不同。
