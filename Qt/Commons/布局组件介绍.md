<center class="my_markdown"><b class="my_markdown">表2-4　组件面板上用于布局的组件</b></center>

| 布局组件                                      | 功能                                                         |
| :-------------------------------------------- | :----------------------------------------------------------- |
| <img src="./images/09.png"/>Vertical Layout   | 垂直方向布局，组件自动在垂直方向上分布                       |
| <img src="./images/10.png"/>Horizontal Layout | 水平方向布局，组件自动在水平方向上分布                       |
| <img src="./images/11.png"/>Grid Layout       | 网格状布局，网状布局大小改变时，每个网格的大小都改变         |
| <img src="./images/12.png"/>Form Layout       | 窗体布局，与网格状布局类似，但是只有最右侧的一列网格会改变大小 |
| <img src="./images/13.png"/>Horizontal Spacer | 一个用于水平分隔的空格                                       |
| <img src="./images/14.png"/>Vertical Spacer   | 一个用于垂直分隔的空格                                       |
