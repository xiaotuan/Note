余数运算符 (a % b) 计算出 a 中有多少 b 的倍数，并返回剩余的值（称为余数）。

>   注意：
>
>   余数运算符 ( `%` ) 在其他语言中也称为*模运算符*。然而，它在 Swift 中对负数的行为意味着，严格来说，它是余数而不是模运算。

余数运算符的工作原理如下。要计算 `9 % 4`，您首先要算出 9 里面可以装多少 4：

![01](./images/01.png)

您可以在 `9` 中放入两个 `4`，剩下的是 `1`（以橙色显示）。

在 Swift 中，这将被写成：

```swift
9 % 4    // equals 1
```

为确定 `a % b` 的答案，`%` 运算符计算以下等式并将 `remainder` 其作为输出返回：

```
a = (b x some multiplier) + remainder
```

其中 `some multiplier` 是 a 除以 b 的最大倍数。

在计算负值的余数时应用相同的方法：

```swift
-9 % 4   // equals -1
```

将 `-9` 和 `4` 插入到等式中得到：

````
-9 = (4 x -2) + -1
````

得出余值为 -1。

对于 `b` 为负值的情况，忽略 `b` 的符号。这意味着 `a % b` 和 `a % -b` 的结果是一样的。
