### 2. 终止进程

```shell
$ kill PROCESS_ID_LIST
```

`kill` 命令默认发出一个 `TERM` 信号。进程 ID 列表使用空格作为定界符。

### 3. 通过 kill 命令向进程发送指定的信号

```shell
$ kill -s SIGNAL PID
```

参数 SIGNAL 要么是信号名称，要么是信号编号。常用的信号如下所示：

+ SIGHUP 1 —— 对控制进程或终端的中介进行挂起检测。
+ SIGINT 2 —— 当按下 <kbd>Ctrl</kbd>+<kbd>C</kbd> 时发送该信号。
+ SIGKILL 9 —— 用于强行杀死进程。
+ SIGTERM 15 —— 默认用于终止进程。
+ SIGTSTP 20 —— 当按下 <kbd>Ctrl</kbd>+<kbd>Z</kbd> 时发送该信号。

### 4. 强行杀死进程

```shell
$ kill -s SIGKILL PROCESS_ID
```

或者

```shell
$ kill -9 PROCESS_ID
```
