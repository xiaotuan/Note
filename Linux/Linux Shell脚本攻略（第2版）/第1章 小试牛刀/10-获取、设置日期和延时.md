1. 读取日期：

```shell
$ date
Thu May 20 23:09:04 IST 2010
```

2. 打印纪元时：

```shell
$ date +%s
1290047248
```

要将日期字符串转换成纪元时，只需要这样即可实现：

```shell
$ date --date "Thu Nov 18 08:07:21 IST 2010" +%s
1290047841
```

例如：

```shell
$ date --date "Jan 20 2001" +%A
Saturday
```

3. 用格式串结合 + 作为 date 命令的参数，可以按照你的选择打印出对应格式的日期。例如：

```shell
$ date "+%d %B %Y"
20 May 2010
```

4. 设置日期和时间：

```shell
# date -s "格式化的日期字符串"
```

例如：

```shell
# date -s "21 June 2009 11:01:22"
```
