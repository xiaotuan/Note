### 1. 打印十进制

十进制使用 `%d` 转换说明符。

### 2. 打印八进制

八进制使用 `%o` 转换说明符，如果希望显示前缀 `0` ，可以使用 `%#o` 转换说明符。

### 3. 打印十六进制

十六进制使用 `%x` 转换说明符，如果希望显示 `0x` 可以使用 `%#x` 转换说明符；如果希望显示 `0X` 可以使用 `%#X` 转换说明符。

```c
/* bases.c -- 以十进制、八进制、十六进制打印十进制数100 */
#include <stdio.h>

int main(void)
{
    int x = 100;

    printf("dec = %d; octal = %o; hex = %x\n", x, x, x);
    printf("dec = %d; octal = %#o; hex = %#x\n", x, x, x);

    return 0;
}
```

输出结果如下：

```
dec = 100; octal = 144; hex = 64
dec = 100; octal = 0144; hex = 0x64
```

### 4. 打印 unsigned in 类型的值

打印 `unsigned int` 类型的值，可以使用 `%u` 转换说明符。

### 5. 打印 long 类型的值

打印 `long` 类型的值，可以使用 `%ld` 转换说明符。

### 6. 以十六进制格式打印 long 类型整数

可以使用 `%lx` 打印十六进制显示的 `long` 类型整数。

### 7. 以八进制格式打印 long 类型整数

可以使用 `%lo` 打印八进制显示的 `long` 类型整数。

### 8. 其他

对于 `short` 类型，可以使用 `h` 前缀。`%hd` 表示以十进制显示 `short` 类型的整数，`%ho` 表示以八进制显示 `short` 类型的整数。`h` 和 `l` 前缀都可以和 `u` 一起使用，用于表示无符号类型。例如，`%lu` 表示打印 `unsigned long` 类型的值。
