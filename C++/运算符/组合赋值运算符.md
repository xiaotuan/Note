<center><b>组合赋值运算符</b></center>

| 操作符 | 作用（L 为左操作数，R 为右操作数） |
| ------ | ---------------------------------- |
| `+=`   | 将 L + R 赋给 L                    |
| `-=`   | 将 L - R 赋给 L                    |
| `*=`   | 将 L * R 赋给 L                    |
| `/=`   | 将 L / R 赋给 L                    |
| `%=`   | 将 L % R 赋给 L                    |
