将递增递减运算符用于指针时，将把指针的值增加（或减少）其指向的数据类型占用的字节数。例如：

```cpp
double arr[5] = { 21.1, 32.8, 23.4, 45.2, 37.4 };
double *pt = arr;	// pt points to arr[0], i.e. to 21.1
++pt;	// pt points to arr[1], i.e. to 32.8
```

也可以结合使用这些运算符和 `*` 运算符来修改指针指向的值。将 `*` 和 `++` 、`--` 同时用于指针时提出了这样的问题：将什么解除，将什么递增。这取决于运算符的位置和优先级。前缀递增、前缀递减和解除引用运算符的优先级相同，以从右到左的方式进行结合。后缀递增和后缀递减的优先级相同，但比前缀运算符的优先级高，这两个运算符以从左到右的方式进行结合。

前缀运算符的从右到左的结合规则意味着 `*++pt` 的含义如下：先将 `++` 应用于 `pt`（因为 `++` 位于 `*` 的右边），然后将 `*` 应用于被递增后的 `pt`：

```cpp
double x = *++pt;	// increment pointer, take the value; i.e, arr[2], or 23.4
```

另一方面，`++*pt` 意味着先取得 `pt` 指向的值，然后将这个值加 1：

```cpp
++*pt;	// increment the pointed to value; i.e., change 23.4 to 24.4
```
