### 1.2　安装Rust工具链

Rust工具链包含两个主要组件：编译器rustc和软件包管理器Cargo，后者有助于管理Rust项目。工具链的发布有3个渠道。

+ **夜间版（** Nightly）：主开发分支每天成功构建的程序。这包括最新的功能，其中许多功能不是很稳定。
+ **测试版（** Beta）：该版本每6周发布一次，一个新的测试分支从夜间版开始。它仅包含已经稳定的功能。
+ **稳定版（** Stable）：该版本每6周发布一次。之前的测试分支会演变成新的稳定版本。

建议开发人员使用稳定版的工具链。但是，夜间版可以实现某些前沿特性，某些库和程序需要用到它。你可以通过rustup轻松地修改夜间版的工具链。
