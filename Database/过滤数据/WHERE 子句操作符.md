<center><b>WHERE 子句操作符</b></center>

| 操作符    | 说明               |
| --------- | ------------------ |
| `=`       | 等于               |
| `<>`      | 不等于             |
| `!=`      | 不等于             |
| `<`       | 小于               |
| `<=`      | 小于等于           |
| `!<`      | 不小于             |
| `>`       | 大于               |
| `>=`      | 大于等于           |
| `!>`      | 不大于             |
| `BETWEEN` | 在指定的两个值之间 |
| `IS NULL` | 为 `NULL` 值       |
