计算字段的另一常见用途是对检索出的数据进行算术计算。例如：

```sql
mysql> SELECT prod_id, quantity, item_price, quantity * item_price AS expanded_price FROM OrderItems WHERE order_num = 20008;
+---------+----------+------------+----------------+
| prod_id | quantity | item_price | expanded_price |
+---------+----------+------------+----------------+
| RGAN01  |        5 |       4.99 |          24.95 |
| BR03    |        5 |      11.99 |          59.95 |
| BNBG01  |       10 |       3.49 |          34.90 |
| BNBG02  |       10 |       3.49 |          34.90 |
| BNBG03  |       10 |       3.49 |          34.90 |
+---------+----------+------------+----------------+
5 rows in set (0.01 sec)

```

下表中列出的 `SQL` 支持的基本算术操作符。此外，圆括号可用来区分优先顺序。

<center><b>SQL 算术操作符</b></center>

| 操作符 | 说明 |
| ------ | ---- |
| +      | 加   |
| -      | 减   |
| *      | 乘   |
| /      | 除   |

> 提示：`SELECT` 语句为测试、检验函数和计算提供了很好的方法。虽然 `SELECT` 通常用于从表中检索数据，但是省略了 `FROM` 子句后就是简单地访问和处理表达式，例如 `SELECT 3 * 2;` 将返回 6;
