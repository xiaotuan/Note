`SIGN(x)` 返回参数的符号，`x` 的值为负、零或正时返回结果依次为 -1、0 或 1。

```sql
mysql> SELECT SIGN(-21), SIGN(0), SIGN(21);
+-----------+---------+----------+
| SIGN(-21) | SIGN(0) | SIGN(21) |
+-----------+---------+----------+
|        -1 |       0 |        1 |
+-----------+---------+----------+
1 row in set (0.00 sec)
```
