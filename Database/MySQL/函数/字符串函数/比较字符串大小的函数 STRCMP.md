`STRCMP(s1, s2)`：若所有的字符串均相同，则返回 0；若根据当前分类次序，第一个参数小于第二个，则返回 -1；其他情况返回 1。

```sql
mysql> SELECT STRCMP('txt', 'txt2'), STRCMP('txt2', 'txt'), STRCMP('txt', 'txt');
+-----------------------+-----------------------+----------------------+
| STRCMP('txt', 'txt2') | STRCMP('txt2', 'txt') | STRCMP('txt', 'txt') |
+-----------------------+-----------------------+----------------------+
|                    -1 |                     1 |                    0 |
+-----------------------+-----------------------+----------------------+
1 row in set (0.00 sec)
```
