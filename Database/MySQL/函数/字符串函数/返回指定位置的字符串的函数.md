`ELT(N, 字符串1, 字符串2, 字符串3, ..., 字符串N)`：若 `N = 1`，则返回值为字符串1；若 `N = 2`，则返回值为字符串2；以此类推；若 `N` 小于或大于参数的数目，则返回值为 `NULL`。

```sql
mysql> SELECT ELT(3, '1st', '2nd', '3rd'), ELT(3, 'net', 'oa');
+-----------------------------+---------------------+
| ELT(3, '1st', '2nd', '3rd') | ELT(3, 'net', 'oa') |
+-----------------------------+---------------------+
| 3rd                         | NULL                |
+-----------------------------+---------------------+
1 row in set (0.00 sec)
```
