`SPACE(n)` 返回一个由 `n` 个空格组成的字符串：

```sql
mysql> SELECT CONCAT('(', SPACE(6), ')');
+----------------------------+
| CONCAT('(', SPACE(6), ')') |
+----------------------------+
| (      )                   |
+----------------------------+
1 row in set (0.00 sec)
```

`REPLACE(s, s1, s2)` 使用字符串 `s2` 替代字符串 `s` 中所有的字符串 `s1`：

```sql
mysql> SELECT REPLACE('xxx.mysql.com', 'x', 'w');
+------------------------------------+
| REPLACE('xxx.mysql.com', 'x', 'w') |
+------------------------------------+
| www.mysql.com                      |
+------------------------------------+
1 row in set (0.00 sec)
```
