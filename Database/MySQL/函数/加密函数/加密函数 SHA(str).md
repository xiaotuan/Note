`SHA(str)` 从原明文密码 `str` 计算并返回加密后的密码字符串，当参数为 `NULL` 时，返回 `NULL`。`SHA` 加密算法比 `MD5` 更安全。

```sql
mysql> SELECT SHA('tom123456');
+------------------------------------------+
| SHA('tom123456')                         |
+------------------------------------------+
| 8218b487f490cb484f45c31403eb1f597a2b531a |
+------------------------------------------+
1 row in set (0.00 sec)
```
