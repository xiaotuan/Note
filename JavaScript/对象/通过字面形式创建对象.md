要用对象字面形式创建对象，可以在大括号内定义一个新对象及其属性。属性的组成包括一个标识符或字符串、一个冒号以及一个值。多个属性之间用逗号分隔。例如：

```js
var book = {
  name: "The Principles of Ojbect-Oriented JavaScript",
  year: 2014
};
```

属性名字也可以用字符串表示，特别是当你希望名字中包括空格或其他特殊字符时。

```js
var book = {
  "name": "The Principles of Ojbect-Oriented JavaScript",
	"year": 2014
};
```
