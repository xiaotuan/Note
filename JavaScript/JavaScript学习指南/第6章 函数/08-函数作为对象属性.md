### 6.4　函数作为对象属性

当函数作为一个对象的属性时，通常被称为方法，从而跟一般函数区分开来（稍后会介绍更多关于函数和方法之间的细微差别）。前面已在第3章中学习了如何将方法添加到一个已有的对象中。也可以在对象初始化的时候添加方法：

```javascript
const o = {
   name: 'Wallace',                       // 简单属性
   bark: function() { return 'Woof!'; },  // 函数属性 (或者方法属性)
} 
```

ES6为了方法引入了一个新的快捷语法。以下代码等效于上面的例子：

```javascript
const o = {
   name: 'Wallace',                      //简单属性
   bark() { return 'Woof!'; },           //函数属性 (或者方法属性)
} 
```
