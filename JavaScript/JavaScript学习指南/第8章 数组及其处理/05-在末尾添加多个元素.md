### 8.2.2　在末尾添加多个元素

`concat` 方法可以给数组添加多个元素并返回新数组的拷贝。如果给 `concat` 传一些数组作为参数，它会把这些数组拆分再把拆分后的元素添加到原始数组中。例如：

```javascript
const arr = [1, 2, 3];           
arr.concat(4, 5, 6);           // 返回[1, 2, 3, 4, 5, 6]; arr 没有改动
arr.concat([4, 5, 6]);         // 返回 [1, 2, 3, 4, 5, 6]; arr没有改动
arr.concat([4, 5], 6);         // 返回 [1, 2, 3, 4, 5, 6]; arr没有改动
arr.concat([4, [5, 6]]);       // 返回 [1, 2, 3, 4, [5, 6]]; arr没有改动
```

需要注意的是， `concat` 只会拆分直接传给它的数组；并不会拆分数组内的数组。
