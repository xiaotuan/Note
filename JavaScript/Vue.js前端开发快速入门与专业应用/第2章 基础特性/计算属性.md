`Vue.js` 还提供了计算属性这种方法，避免在模板中加入过重的业务逻辑，保证模板的结构清晰和可维护性。

**1. 基础例子**

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <title>Computed Attributes</title>
    <script src="https://cdn.jsdelivr.net/npm/vue@2.6.11"></script>
  </head>
  <body>
    <div id="app">
      <p>{{firstName}}</p>
      <p>{{lastName}}</p>
      <p>{{fullName}}</p>
    </div>
    <script>
      var vm = new Vue({
        el: '#app',
        data: {
          firstName: 'Gavin',
          lastName: 'CLY'
        },
        computed: {
          fullName: function() {
            // this 指向 vm 实例
            return this.firstName + ' ' + this.lastName
          }
        }
      })
    </script>
  </body>
</html>
```

**2. Setter**

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <title>Setter Computed Attributes</title>
    <script src="https://cdn.jsdelivr.net/npm/vue@2.6.11"></script>
  </head>
  <body>
    <div id="el">
      <p>&yen;{{price}}</p>
    </div>
    <script>
      var vm = new Vue({
        el: '#el',
        data: {
          cents: 100,
        },
        computed: {
          price: {
            set: function(newValue) {
              this.cents = newValue * 100
            },
            get: function() {
              return (this.cents / 100).toFixed(2)
            }
          }
        }
      })
    </script>
```
