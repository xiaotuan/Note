<center><font size="5"><b>自定义组件监听数据改变</b></font></center>

**1. 定义属性**

```js
Component({
  data: {
    list: [1, 2, 3, 4, 5]
  },
  properties: {
    propA: {
      type: Number,
      value: 0,
    },
    propB: {
      type: Number,
      value: 0,
    },
  }
}
```

**2. 监听属性数据改变**

```js
Component({
  data: {
    list: [1, 2, 3, 4, 5]
  },
  properties: {
    propA: {
      type: Number,
      value: 0,
    },
    propB: {
      type: Number,
      value: 0,
    },
  },
  lifetimes: {
    // 生命周期函数，可以为函数，或一个在methods段中定义的方法名
    attached: function () { 
      this.setData({
        list: [6, 7, 8, 9, 0]
      })
    },
  },
  observers: {
    'propA, propB': function (propA, propB) {
      console.log("A: " + propA + ", B: " + propB)
    },
    'list': function(list) {
      console.log(list)
    }
  }
})
```

**3. 另一种监听数据改变的方法**

```js
Component({
  data: {
    list: [1, 2, 3, 4, 5]
  },
  properties: {
    propA: {
      type: Number,
      value: 0,
      observer() {
        console.log(this.__data__.propA)
      }
    },
    propB: {
      type: Number,
      value: 0,
      observer() {
        console.log(this.__data__.propB)
      }
    },
  }
}
```
