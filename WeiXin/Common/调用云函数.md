<center><font size="5"><b>调用云函数</b></font></center>

```js
wx.cloud.callFunction({
  // 要调用的云函数名称
  name: 'add',
  // 传递给云函数的参数
  data: {
    x: 1,
    y: 2,
  },
  success: res => {
    // output: res.result === 3
  },
  fail: err => {
    // handle error
  },
  complete: () => {
    // ...
  }
})
```

+ name：云函数的名称
+ data：传递给云函数的参数
+ success：调用成功的回调函数
+ fail：调用失败的回调函数
+ complete：调用完成的回调函数

```js
wx.cloud.callFunction({
  // 要调用的云函数名称
  name: 'add',
  // 传递给云函数的event参数
  data: {
    x: 1,
    y: 2,
  }
}).then(res => {
  // output: res.result === 3
}).catch(err => {
  // handle error
})
```
