<center><font size="5"><b>fixed 固定定位的使用</b></font></center>

**使用方法**

```css
{
    position: fixed;
}
```

固定定位和绝对定位非常类似，不过被定位的容器不会随着滚动条的拖动而变化位置。在视野中，固定定位的容器的位置是不会改变的。

> 打个比方：如果在界面上设置了一个固定定位的元素，而界面的其他元素的高度或宽度超出了浏览器的宽度或高度，这时我们可以通过拖动滚动条来查看超出部分的内容，但是在拖动滚动条时，固定定位的元素的位置不会改变，也就是说拖动滚动条不能使用固定定位的元素滚出浏览器内容区域外或滚入浏览器内容区域中。

> 注意：固定定位不支持 `z-index` 属性。
