<center><font size="5"><b>文本加粗</b></font></center>

可以通过 `font-weight` 属性设置文字的加粗效果，可用的属性值有：

| 值                          | 描述                                                        |
| :-------------------------- | :---------------------------------------------------------- |
| normal                      | 默认值。定义标准的字符。                                    |
| bold                        | 定义粗体字符。                                              |
| bolder                      | 定义更粗的字符。                                            |
| lighter                     | 定义更细的字符。                                            |
| 100200300400500600700800900 | 定义由粗到细的字符。400 等同于 normal，而 700 等同于 bold。 |
| inherit                     | 规定应该从父元素继承字体的粗细。                            |
