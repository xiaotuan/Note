<center><font size="5"><b>relative 相对定位的使用</b></font></center>

**使用方法**

```css
{
    position: relative;
}
```

如果 对 一个 元素 进行 相对 定位， 它将 出 现在 它 所在 的 位置 上。 然后， 可以 通过 设置 垂直 或 水平 位置， 让 这个 元素“ 相对于” 它的 起点 进行 移动。

当 容器 的 position 属性 值 为 relative 时， 这个 容器 即被 相对 定位 了。 相对 定位 和 其他 定位 相似， 也是 独立 出来 浮在 上面。 不过 相对 定位 的 容器 的 top（ 顶部）、 bottom（ 底部）、 left（ 左边） 和 right（ 右边） 属性 参照 对象 是 其父 容器 的 4 条 边， 而 不是 浏览器 窗口。

> 注意：
>
> 1. 元素使用相对定位后，设置 `left`、`right`、`top`、`bottom` 和 `margin` 的值时是相对于它的原始位置而言的。
> 2. 元素使用相对定位后，该元素原先的位置，父容器的其他子元素是无法占用的其原先的位置，也就是该位置会空在哪里。
> 3. 相对 定位 的 容器 其实 并未 完全 独立， 其 浮动 范围 仍然 在 父 容器 内， 并且 其 所占 的空白 位置 仍然 有效地 存在 于 前后 两个 容器 之间。
> 4. absolute， CSS 中的 写法 是 position: absolute， 意思是 绝对 定位， 它是 参照 浏览器 的 左上 角， 配合 top、 right、 bottom、 left 进行 定位。relative， CSS 中的 写法 是 position: relative， 意思是 相对 定位， 它是 参照 父 级 的 原 始点 为 原始 点， 无父 级 则以 文本 流的 顺序 在上 一个 元素 的 底部 为 原始 点， 配合 top、 right、 bottom、 left 进行 定位。
