<center><font size="5"><b>指定文本显示行数和超出省略显示</b></font></center>

显示文本的组件必须是 `text` ，实现效果的 `css` 样式如下：

```css
.up-title {
  color: #0B0B0B;
  font-size: 28rpx;
  font-weight: 500;
    
  display: -webkit-box;
  -webkit-box-orient: vertical;
  -webkit-line-clamp:2;
  overflow: hidden;
  word-break: break-all;
  text-overflow: ellipsis;
}
```

其中 `-webkit-line-clamp` 属性的值为要显示的最大行数。
