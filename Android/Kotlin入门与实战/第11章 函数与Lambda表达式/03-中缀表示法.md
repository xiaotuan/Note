### 11.1.2　中缀表示法

所谓中缀表达式，就是将函数名放到两个操作数中间的表达式，其中，左侧的操作数代表函数对象或值，右侧的操作数代表函数的参数值。在Kotlin中，只有满足以下几个条件的函数才能被中缀表示法调用。

+ 函数为成员函数或扩展函数。
+ 函数只有一个参数。
+ 函数用infix关键字标记。

下面举一个简单的例子来说明中缀表示法是如何调用函数的。

```python
//Int的扩展函数
infix fun Int.shl(x:Int):Int{
    return x;
}
fun main(args: Array<String>) {
    //中缀表示法调用函数
    println(1 shl 2)     // 输出4
}
```
