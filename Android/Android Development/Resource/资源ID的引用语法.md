将 `id` 分配给 XML 文件中资源的语法称为**资源引用语法**。正确的语法结构如下：

```
@[package:]type/name
```

`type` 对应于 `R.java` 中一种可用的资源类型命名空间，包括：

+ `R.drawable`
+ `R.id`
+ `R.layout`
+ `R.string`
+ `R.attr`
+ `R.plural`
+ `R.array`

`XML` 资源引用语法中相应的类型为：

+ `Drawable`
+ `id`
+ `layout`
+ `string`
+ `attr`
+ `plurals`
+ `string-array`
