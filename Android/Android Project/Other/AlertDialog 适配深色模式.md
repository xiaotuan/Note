### 1. 第三方应用

深色模式使用如下样式：

```xml
<style name="Theme.AlertDialog" parent="@style/ThemeOverlay.MaterialComponents.Dialog.Alert.Framework">
</style>
```

非深色模式使用如下样式：

```xml
<style name="Theme.AlertDialog" parent="@style/ThemeOverlay.MaterialComponents.Light.Dialog.Alert.Framework">
</style>
```

### 2. 系统应用

深色模式使用如下样式：

```xml
<style name="Theme.AlertDialog" parent="@android:style/Theme.Material.Light.Dialog.Alert">
</style>
```

非深色模式使用如下样式：

```xml
<style name="Theme.AlertDialog" parent="@android:style/Theme.Material.Dialog.Alert">
</style>
```
