### 1. 显示所有远程仓库

```console
$ git remote -v
```

### 2. 显示某个远程仓库的信息

```console
$ git remote show [remote]
```

### 3. 添加远程版本库：

```console
$ git remote add [shortname] [url]
```

### 4. 删除远程版本库

```console
$ git remote rm [name]
```

### 5. 修改远程版本库名称

```console
$ git remote rename old_name new_name
```
