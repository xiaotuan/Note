### 1. Debian/Ubuntu

下面的命令通过以 `root` 权限执行 `apt-get` 来安装重要的 `Git` 包：

```shell
$ sudo apt-get install git git-doc gitweb git-gui gitk git-email git-svn
```

### 2.其他发行版

在 `Gentoo` 系统中，使用 `emerge` 命令：

```shell
$ sudo emerge dev-util/git
```

在 `Fedora` 中，使用 `yum` 命令：

```shell
$ sudo yum install git
```
