### 结果分析

`ThreadFactory` 接口只有一个名为 `newThread()` 的方法。该方法接收一个 `Runnable` 对象作为参数，并返回一个 `Thread` 对象。实现一个 `ThreadFactory` 接口时，必须覆盖 `newThread()` 方法。实现最基本的 `ThreadFactory` 仅需一行代码：

```css
return newThread(r);
```

+ 创建个性化线程。如本案例所示，使用特殊格式来命名，甚至可以通过继承 `Thread` 类来创建自己的线程类。
+ 如前面案例所示，保存创建线程的统计信息。

+ 验证线程的创建。
