| 运算符                                                       | 结合性   |
| ------------------------------------------------------------ | -------- |
| [ ]、.、( )（方法调用）                                      | 从左向右 |
| !、~、++、--、+（一元运算符）、-（一元运算符）、( )（强制类型转换）、new | 从右向左 |
| *、/、%                                                      | 从左向右 |
| +、-                                                         | 从左向右 |
| <<、>>、>>>                                                  | 从左向右 |
| <、<=、>、>=、instanceof                                     | 从左向右 |
| ==、!=                                                       | 从左向右 |
| &                                                            | 从左向右 |
| ^                                                            | 从左向右 |
| \|                                                           | 从左向右 |
| &&                                                           | 从左向右 |
| \|\|                                                         | 从左向右 |
| ?:                                                           | 从右向左 |
| =、+=、-=、*=、/=、%=、&=、\|=、^=、<<=、>>=、>>>=           | 从右向左 |
