<center><b>常用的转义字符及其含义</b></center>

| 转义字符 | 转义字符的意义                 | ASCII 码 |
| -------- | ------------------------------ | -------- |
| `\n`     | 回车换行                       | 10       |
| `\t`     | 横向跳到下一个制表位置         | 9        |
| `\b`     | 退格                           | 8        |
| `\r`     | 回车                           | 13       |
| `\f`     | 走纸换页                       | 12       |
| `\\`     | 反斜杠                         | 92       |
| `\'`     | 单引号                         | 39       |
| `\"`     | 双引号                         | 34       |
| `\a`     | 鸣铃                           | 7        |
| `\ddd`   | 1 ~ 3 位八进制数所代表的字符   |          |
| `\xhh`   | 1 ~ 2 位十六进制数所代表的字符 |          |
