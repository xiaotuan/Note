程序清单 **2.2 fathm_ft.c** 程序

```c
// fathm_ft.c -- 把2音寻转换成英寸
#include <stdio.h>
int main(void) 
{
	int feet, fathoms;
	fathoms = 2;
	feet = 6 * fathoms;
	printf("There are %d feet in %d fathoms!\n", feet, fathoms);
	printf("Yes, I said %d feet!\n", 6 * fathoms);
	return 0;
}
```

程序在一条声明中声明了两个变量，需要在声明中用逗号隔开两个变量。
