程序清单 **3.10 escape.c** 程序

```c
/* escape.c -- 使用转义序列 */
#include <stdio.h>

int main(void)
{
	float salary;
	printf("\aEnter your desired monthly salary:");	/* 1 */
	printf("$________\b\b\b\b\b\b\b\b");	/* 2 */
	scanf("%f", &salary);
	printf("\n\t$%.2f a month is $%.2f a year.", salary, salary * 12.0);	/* 3 */
	printf("\rGee!\n");	/* 4 */
	return 0;
}
```

程序输出如下：

```console
Enter your desired monthly salary:$4000.000

Gee!	$4000.00 a month is $48000.00 a year.
```
