<center><b>运算符优先级</b></center>

| 运算符                           | 结合律   |
| -------------------------------- | -------- |
| `()`                             | 从左往右 |
| `+`、`-`（一元）、++、--、sizeof | 从右往左 |
| `*`、`/`、%                      | 从左往右 |
| `+`、`-`（二元）                 | 从左往右 |
| >、<、<=、>=                     | 从左往右 |
| ==、!=                           | 从左往右 |
| `=`                              | 从右往左 |
