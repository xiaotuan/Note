在浏览网页时经常会看到一些欢迎信息的界面，在经过一段时间后，这一页面会自动调整到其他页面中，这就是网页的跳转。

**语法**

```html
<meta http-equiv="refresh" content="跳转时间;url=链接地址" />
```

> 注意
>
> 跳转时间的单位为秒。

**示例**

```html
<!DOCTYPE html>
<html>
    <head>
        <title>设置网页定时跳转</title>
        <meta http-equiv="refresh" content="3;url=https://www.baidu.com" />
    </head>
    <body>
        
    </body>
</html>
```
