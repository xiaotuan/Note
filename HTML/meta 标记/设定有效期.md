在某些网站上会设置网页的到期时间，一旦过期则必须到服务器上重新调用。

**语法**

```html
<meta http-equiv="expires" content="到期的时间" />
```

> 注意
>
> 到期的时间必须是 GMT 时间格式，即 `星期,日月年时分秒`，这些时间都是要英文和数字进行设定。

**示例**

```html
<!DOCTYPE html>
<html>
    <head>
        <title>设置网页有效期</title>
        <meta http-equiv="expires" content="Web,14 september 2022 11:35:00 GMT" />
    </head>
    <body>
        
    </body>
</html>
```
