设置页面关键字是为了向搜索引擎说明这一网页的关键字，从而帮助搜索引擎对该网页进行查找和分类，一般可设置多个关键字，关键字之间使用逗号隔开。

> 注意
>
> 由于很多搜索引擎在检索时会限制关键字的数量，因此在设置关键字时不要过多。

`<meta>` 标记语法如下：

`<meta name="keyname" content="具体的关键字" />`

例如：

```html
<!DOCTYPE html>
<html>
    <head>
        <title>设置页面关键字</title>
        <meta name="keyword" content="html,元信息,关键字"/>
    </head>
    <body>
        
    </body>
</html>
```
