**语法**

```html
<meta name="robots" content="搜索方式" />
```

<center><b>content 值与其对应的含义</b></center>

| Content 的值 | 描述                               |
| ------------ | ---------------------------------- |
| All          | 表示能搜索当前网页及其链接的网页   |
| Index        | 表示能搜索当前网页                 |
| Nofollow     | 表示不能搜索与当前网页链接的网页   |
| Noindex      | 表示不能搜索当前网页               |
| None         | 表示不能搜索当前网页及其链接的网页 |

**示例**

```html
<!DOCTYPE html>
<html>
    <head>
        <title>限制搜索方式</title>
        <meta name="robots" content="index"/>
    </head>
    <body>
        
    </body>
</html>
```
