#### 
  9.4.2 自定义函数的调用


在MySQL中，自定义函数的调用方法与MySQL内部函数的调用方法是相同的。用户自己定义的函数与MySQL内部函数性质相同。区别在于，自定义函数是用户自己定义的，而内部函数是MySQL的开发者定义的。

【范例9-12】

调用范例9-10中的自定义函数name_emp，代码如下。

&#13;
    Select name_emp(2);&#13;
    mysql> select name_emp(1);&#13;
    +-------------+&#13;
    | name_emp(1) |&#13;
    +-------------+&#13;
    |张丽    |&#13;
    +-------------+&#13;
    1 row in set (0.00 sec)&#13;
