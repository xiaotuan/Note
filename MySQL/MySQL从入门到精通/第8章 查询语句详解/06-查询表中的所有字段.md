#### 
  8.2.3 查询表中的所有字段


将表中所有的字段都查询出有两种方法：一种是在SELECT子句中将所有的字段都列出；另一种是将SELECT子句中的的属性列表指定为“*”,该符号代表所有列。

技巧 
 如果表中的字段很多，使用“*”后查询的数据量很大，会降低数据的传输效率。所以，除非需要查询表中的所有字段，最好不用“*”。

【范例8-2】

查询所有学生的记录。

&#13;
    MySQL> select *&#13;
    -> from student;&#13;

给语句等价于如下语句。

&#13;
    MySQL> select sno,sname,ssex,sage,sdept&#13;
    -> from student;&#13;
